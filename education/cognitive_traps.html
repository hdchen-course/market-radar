交易認知陷阱 — 你的大腦在騙你

15 個最致命的決策偏誤 + 30 題情境訓練 — 每次虧損背後都有一個認知 Bug

為什麼需要這篇：你不是不會賺（4個月+$150K證明了）。你虧的每一分錢都來自「大腦的系統性判斷錯誤」— 不是情緒失控，而是思考本身有 Bug。這些 Bug 是人類演化留下的，每個人都有，但可以透過覺察和訓練來規避。

致命陷阱 #1 ~ #5：持倉時的判斷扭曲

🧠 陷阱 #1：處置效應（Disposition Effect）

定義：急著賣贏的、死抱虧的。賺一點就跑，虧很多才走。

原因：人對「確定的小收益」的快樂 > 「可能的大收益」的期望。但對虧損，人寧可賭一把「可能回本」也不願接受「確定的小虧損」。

你的真實案例：白銀從 $72.28 開始虧 → 到 $67 該走（設好的止損）→ 沒走（不想確定虧損）→ 最後在 $61.53 被爆倉。同時你前面賺的單子都是「$20-50 就跑」。
= 贏的跑太快（$20-50），虧的扛太久（$72→$61）。
解藥：
  • 開倉前就寫下：「這筆我最多虧 $___，到了就走。」
  • 把止損單掛上去，不取消。讓系統替你做決定，繞過大腦。
  • 反過來想：如果你現在是空手，會不會在這個價位開多？如果不會 → 那你為什麼還抱著？

🧠 陷阱 #2：沉沒成本謬誤（Sunk Cost Fallacy）

定義：「已經虧這麼多了，不甘心現在走」— 用過去的投入來合理化繼續持有。

原因：大腦不想承認之前的決定是錯的。平倉 = 承認失敗。所以繼續抱 = 「還沒失敗」的幻覺。

你的案例：白銀成本 $72.28，跌到 $67（止損線）時心想：「已經虧了 $5/顆，如果現在走就確定虧了。萬一明天漲回來呢？」→ 結果跌到 $61.53。

正確思考：「我虧了 $5 是已經發生的事實，跟未來方向無關。問題只有一個：從現在 $67 開始，白銀更可能漲還是跌？」
解藥 — 「Clean Sheet」思維：
  • 假裝你現在是空手。你會在這個價位開這個方向嗎？
  • 如果答案是 No → 立刻平倉。你的成本跟未來方向毫無關係。
  • 口訣：「成本是歷史，價格是現在，方向是未來。三者無關。」

🧠 陷阱 #3：錨定效應（Anchoring）

定義：被一個「參考數字」綁住判斷。例如被自己的成本價、前高、或目標價錨定。

原因：大腦需要一個「定錨」來做判斷。一旦設定，很難擺脫。

你的案例（多次出現）：
  • 錨定成本：「$72.28 進的，至少要等到 $70 才平」→ 結果 $70 到不了，一路跌到爆倉
  • 錨定目標：設目標 $70 → 到 $68.56 時「再等一下就到了」→ 結果反轉暴跌
  • 錨定前高：「上週還 $75，現在才 $67，太便宜了」→ 但市場不在乎你的「太便宜」
解藥：
  • 問自己：「如果我不知道成本/前高/目標，只看現在的圖和數據，我會做什麼？」
  • 用 SuperTrend 方向代替主觀判斷。ST 不會被錨定。
  • 把目標價視為「建議」而非「必須」。到了附近就開始減倉，不等精確命中。

🧠 陷阱 #4：確認偏誤（Confirmation Bias）

定義：只看支持自己方向的資訊，忽略反面證據。

原因：持有多單時，大腦會自動過濾掉「看空」的資訊，放大「看多」的資訊。

你的案例：白銀三框全多時 → 只看到「趨勢向上、金銀比收斂」→ 忽略了「CPI 4.2%、Fed升息、30Y破5%」這些巨大利空。
結果：SuperTrend 在6小時內從三框全多→雙空。但你選擇性忽略了宏觀面的惡化。
解藥 — 「魔鬼代言人」練習：
  • 每次開倉後，強迫自己寫下 3 個「這筆可能錯的理由」
  • 設定「反轉信號」：不是等跌了才想，是開倉時就定義「什麼信號代表我錯了」
  • 看報告時先看風險警告，再看多頭論述。強制自己先消化負面資訊。

🧠 陷阱 #5：損失厭惡（Loss Aversion）

定義：虧 $1 的痛苦是賺 $1 快樂的 2-2.5 倍。所以人會冒更大風險去「避免確認虧損」。

原因：演化設計 — 在野外，失去食物（餓死）比得到額外食物更重要。但在交易中這個本能是有害的。

你的案例：「我不想讓今天變成虧損日」→ 所以浮虧的單子不平 → 結果小虧變大虧。
解法已找到：目標從「每天獲利」改為「每週獲利」— 允許有虧損日。
解藥：
  • 重新定義「贏」：執行了止損 = 贏（贏在紀律）。虧了 $300 但守住紀律 = 成功的交易。
  • 數學化：一週5天，3天賺+2天虧，但賺的>虧的 = 週正。允許虧損日存在。
  • 口訣：「浮虧就是虧，早認早解脫，明天再賺。」

致命陷阱 #6 ~ #10：連勝後的判斷失靈

🧠 陷阱 #6：熱手謬誤（Hot Hand Fallacy）

定義：連贏幾次後以為自己「手感好」「看盤很準」，開始放鬆紀律、加大倉位。

原因：大腦把隨機的連勝解讀為「我的技術變好了」。實際上市場的隨機性不會因為你之前贏了就減少。

你的案例：連續四天每天 +3%（$20K→$22.5K）→ 第五天開始覺得「手感好」→ 加大倉位進白銀 → CPI衝擊 → 爆倉。

連勝讓你：(1) 放鬆止損 (2) 加大倉位 (3) 覺得自己「看懂了」。三者合一 = 災難。
解藥：
  • 連贏 3 天後，第 4 天倉位減半。不是因為會虧，是因為你的判斷力在「贏」的興奮中下降了。
  • 連贏時問自己：「如果明天突然虧3天的利潤，我能接受嗎？」如果不能 → 你倉位太大了。
  • 設固定規則：無論贏多少天，單筆倉位不超過帳戶的 X%。永遠。

🧠 陷阱 #7：過度自信（Overconfidence）

定義：高估自己預測市場方向的能力。「我看得很準」「我理解市場了」。

原因：賺錢時歸功於自己（技術好）。虧錢時歸咎於外部（市場不正常）。這叫「自利歸因偏誤」。

模式：4個月賺$150K → 覺得自己已經「懂了」→ 放鬆風控 → 3週虧回去+更多。
事實：你賺的那4個月，白銀恰好在明確上漲趨勢中。任何人在那個趨勢中做多都會賺。不全是「你的技術」。
解藥：
  • 把每次獲利歸因為：50% 市場環境 + 30% 紀律 + 20% 技術。不是 100% 自己厲害。
  • 問：「如果把我的操作反過來做，會虧多少？」如果答案是「也不會虧太多」→ 代表市場在送錢，不是你在賺錢。
  • 連贏時最危險。把紀律規則寫在螢幕旁邊，連贏時更要看。

🧠 陷阱 #8：目標漂移（Goal Creep）

定義：原本設好目標，到了之後改成更高的目標。「已經到 $68 了，再等等看能不能到 $70」。

原因：當目標接近時，大腦開始計算「如果繼續走會賺更多」。放棄「確定的利潤」讓人痛苦（跟處置效應相反但同源）。

你的案例（反覆出現）：
  • BTC 有賺沒跑 → 改更高目標 → 最後大虧
  • 白銀到 $68.56（距$70很近）→ 「再等一下」→ 暴跌至 $61.53
  • ETH 川習會後有利潤 → 沒平 → 方向反轉
每一次虧損都有「到了沒跑」的影子。
解藥：
  • 到了就走。不改目標。永遠。這是你最需要內化的一條規則。
  • 到目標先平一半（鎖利潤）。剩一半設 trailing stop。這樣「繼續漲」你也有，「反轉」你也不虧。
  • 開倉時把目標寫在紙上。到了的時候不看盤，直接按照紙上寫的執行。

🧠 陷阱 #9：賭徒謬誤 / 均值回歸幻覺

定義：「已經連跌5天了，今天應該要漲了吧？」— 把隨機事件當作有「欠債」要還。

原因：大腦不理解「趨勢」。在趨勢中，連跌 5 天之後連跌第 6 天的機率跟前5天無關。

案例：白銀從 $75 連跌到 $65 → 「跌這麼多了應該反彈了吧」→ 加倉 → 繼續跌到 $61.53。趨勢不會因為「已經跌很多」就停止。
解藥：
  • 不預測反轉。只在反轉確認後跟隨。（1h/4h ST翻轉 = 確認）
  • 「連跌很多」不是做多理由。「ST翻多」才是做多理由。
  • 口訣：「趨勢的持續時間永遠比你以為的長。」

🧠 陷阱 #10：資訊過載 → 分析癱瘓 or 過度交易

定義：看太多資訊反而無法做決定，或者每個資訊都想反應（頻繁進出）。

原因：現代交易者有無限資訊。每一條新聞都感覺「重要」。大腦無法區分噪音和信號。

解藥：
  • 定義你的「決策輸入」只有3個：SuperTrend方向 + 多空計分 + 重大事件日曆。其他都是噪音。
  • 開盤前做一次分析就好。盤中不再重新分析（除非觸發止損）。
  • 後段小單磨的階段，不看新聞。只看價格和掛單成交。

致命陷阱 #11 ~ #15：出場與風控的判斷錯誤

🧠 陷阱 #11：稟賦效應（Endowment Effect）

定義：你擁有的東西比實際更值錢。持有的倉位比實際更「捨不得」。

案例：白銀多單持有多天 → 開始對它有「感情」→ 變成「我的白銀」→ 更不願意平掉。

解藥：倉位只是數字。不是「你的」。它不認識你。每天問：「如果我現在是空手，我會不會在這個價格建立這個倉位？」

🧠 陷阱 #12：近因偏誤（Recency Bias）

定義：過度看重最近發生的事。「昨天反轉搶單賺了 → 今天再搶一次」。

案例：你說的「反轉搶單第一次有效，第二次出問題」。第一次成功讓大腦覺得「這招有效」，忽略了第二次環境已不同。

解藥：看長期統計，不看最近幾次。「反轉搶單」的長期勝率可能只有40% — 第一次贏只是運氣。規則：每天只做一次。

🧠 陷阱 #13：控制錯覺（Illusion of Control）

定義：以為自己能控制結果。「我看著盤就不會虧」「我手動比止損單好」。

案例：不設止損單（「我會自己手動停損」）→ 凌晨睡著 → 醒來已經崩了。你覺得你能控制，但你控制不了自己會睡著。

解藥：
  • 永遠掛止損單。不依靠「我會手動停損」。你會睡著、會猶豫、會改主意。系統不會。
  • 承認：你無法控制市場方向。你能控制的只有：倉位大小、止損位置、進場時機。

🧠 陷阱 #14：結果偏誤（Outcome Bias）

定義：用結果來判斷決策品質。「攤平最後賺了 → 所以攤平是對的」。

案例：你說「昨天攤平賺了算運氣」— 你自己知道這是運氣。但如果不提醒自己，下次攤平時大腦會說：「上次攤平也成功了啊」。

解藥：判斷決策品質要看「過程」不是「結果」。
  • 好決策+壞結果：設了止損被打到（然後市場真的反轉了）→ 這仍然是好決策。
  • 壞決策+好結果：攤平扛單最後回本 → 這仍然是壞決策（下次可能爆倉）。
  • 只獎勵自己「好過程」，不獎勵「好結果」。

🧠 陷阱 #15：現狀偏誤（Status Quo Bias）

定義：維持現狀感覺比改變安全。「已經持有了，就繼續持有吧」— 即使該平倉。

案例：白銀浮虧 → 「維持現狀（不動）比做一個動作（平倉）容易」→ 不動 = 繼續持有 = 虧更多。不做決定本身就是一個決定（決定繼續承擔風險）。

解藥：把「不動」也視為一個主動選擇。每天問：「我今天是 選擇 繼續持有，還是 懶得 做決定？」如果是後者 → 立刻行動。

情境訓練 — 30 題（反覆練習直到成為反射）

使用方式：每題是一個交易情境。先想你的直覺答案，再看正確答案。如果直覺錯了，找到對應的陷阱編號。每週做一次，直到所有直覺都正確。

情境 1

你白銀多單成本 $72，現在跌到 $67。你設的止損是 $67。價格剛碰到 $67，你在想：「$67 好像是支撐位，也許會彈回來，我先取消止損看看。」

你應該怎麼做？

正確：讓止損執行，不取消。
觸發陷阱：#1 處置效應（不想確認虧損）+ #5 損失厭惡 + #13 控制錯覺（以為手動比系統好）
你說「也許會彈回來」= 確認偏誤（#4），只看到支持你想法的理由。
規則：止損觸發就走。不思考。不分析。走。

情境 2

你連續三天每天賺 3%。今天第四天，你想：「手感好，今天可以加大倉位試試看。」

正確：不加倉。維持原本倉位甚至減半。
觸發陷阱：#6 熱手謬誤 + #7 過度自信
連贏三天不代表第四天也會贏。你的「手感」不是真的。市場不會因為你前三天贏就今天也讓你贏。
規則：連贏3天 → 第4天倉位不變或減半。

情境 3

白銀到了 $69.5，你的目標是 $70。你想：「這麼接近了，不差這 $0.5，等到 $70 再說。」結果開始回落到 $68。你又想：「剛才 $69.5 沒跑，現在 $68 更不想跑。等回到 $69.5 再說。」

正確：$69.5 就應該平一半。沒到 $70 沒關係。
觸發陷阱：#3 錨定效應（被 $70 目標錨定）+ #8 目標漂移（到了附近卻不走）
然後回落到 $68 不走 = #3 又被 $69.5 錨定（「等回到剛才的價」）。
規則：目標附近（±2%）就開始減倉。不等精確命中。

情境 4

CPI 今晚公布。你持有多單浮盈 +2%。你想：「如果 CPI 低，可能再漲 3%。賭一把。」

正確：CPI 前至少平一半，鎖利潤。
觸發陷阱：#7 過度自信（以為能預測CPI結果）+ #8 目標漂移（+2%不夠，想要+5%）
CPI 結果是 50/50 賭博。你有 +2% 的確定利潤 vs「可能+5%也可能-5%」的賭博。
規則：重大事件前鎖利潤。不賭。已經有的利潤 > 可能的利潤。

情境 5

反轉搶單第一次賺了 $200。市場又到了一個你覺得「過度」的位置。你想再搶一次反轉。

正確：不做第二次。收工或改用其他模式。
觸發陷阱：#6 熱手謬誤 + #12 近因偏誤（剛才成功 → 以為下次也會）
你自己觀察到的規律：「反轉搶單只有第一次準。第二次通常出問題。」
規則：反轉搶單每天只做一次。做完收工。

情境 6

你今天早上虧了 $400。現在看到一個「很好的機會」，想把虧損賺回來。

正確：今天不做了。明天再來。
觸發陷阱：#5 損失厭惡（急於把虧損「修復」）+ #7 過度自信（以為這個機會一定會賺）
虧損後的「好機會」通常是大腦在急於報復（revenge trading）。判斷力在虧損後是降低的。
規則：日初虧損 → 當天停止交易。

情境 7

白銀已經從 $75 跌到 $65（-13%）。你想：「跌這麼多了，肯定快反彈了，我來接。」

正確：不接。等 SuperTrend 翻多再進場。
觸發陷阱：#9 賭徒謬誤（跌多了≠會漲）+ #3 錨定（被$75錨定覺得$65「很便宜」）
趨勢可以比你想的持續更久。白銀從 $75 → $65 → $61.53 → 可能更低。
規則：不抄底。只在趨勢確認翻轉後跟隨。

情境 8

你的白銀倉位浮虧 $1000。你想：「加倉攤平，只要漲回一點就回本。」

正確：不攤平。只能減倉或平倉。
觸發陷阱：#2 沉沒成本（已虧$1000→想用更多資金「修復」）+ #13 控制錯覺
攤平 = 在錯誤的方向上加注。如果方向是錯的，加倉只會虧更多更快。
規則：浮虧中永遠不加倉。只能減或平。

情境 9

你看到報告說「三時框全多，計分5/5全多，強烈偏多」。你想全倉做多。

正確：最多用30%倉位。不全倉。
觸發陷阱：#4 確認偏誤（只看到利多忽略風險）+ #7 過度自信
報告再怎麼看多，明天可能出一個 CPI/伊朗/黑天鵝把一切打翻。6/9 白銀就是從「三框全多」6小時內翻空的。
規則：再好的信號，倉位也不超過帳戶30%。因為意外永遠存在。

情境 10

凌晨3點，你的多單還有浮盈 +1.5%。你很累想睡了。你想：「應該沒事吧，明天再看。」

正確：設止損單 or 直接平倉。不帶倉位睡覺。
觸發陷阱：#13 控制錯覺（以為「應該沒事」）+ #15 現狀偏誤（不想動）
你的白銀爆倉就是睡著造成的（凌晨從$65→$61.53，你醒來已經爆了）。
規則：睡覺前必須設止損或平倉。無例外。

上次攤平最後賺了。這次又浮虧了，你想：「上次攤平也成功了，這次再試一次。」

情境 12

你想清倉（直覺告訴你該走了），但看了一下價格：「現在跌了 2%，等回到平盤再走。」

正確：想清倉的那一刻，立刻平一半。不等好價格。

你的直覺「該走了」通常是對的。問題是你想「走得漂亮」→ 結果走不了。

你持有BTC多單，有浮盈。Trump 突然發了一條推文利多 BTC。你想加倉。

情境 14

月底了，你這個月虧 $2000。還有2個交易日。你想拼命做交易把月報酬拉正。

正確：接受這個月是虧損月。正常操作，不加大。
觸發陷阱：#5 損失厭惡（月報酬不想是負）+ #7 過度自信（以為兩天能翻盤）
「拼命」= 加大倉位 + 降低標準 + 放鬆止損。= 虧更多的完美配方。
規則：接受有虧損月。一個月虧$2000 << 一天因為「拼」而虧$5000。

情境 15

你已經達成今天 +3% 目標。但你看到一個「超好的機會」就在眼前。你想再做一筆。

正確：收工。不做。明天還有。

更多情境（16-30）— 快速練習

16. 你聽到一個 KOL 說「白銀要漲到$100」。你很興奮想重倉做多。
→ 陷阱 #4 確認偏誤 + #10 資訊過載。KOL 說的不是交易信號。只看 ST + 計分。

17. 你虧了 $500 後，下一筆想加大倉位「贏一筆大的」回來。
→ 陷阱 #5 損失厭惡（revenge trading）。虧損後縮小倉位而非加大。

18. SuperTrend 翻空了，但你覺得「它會翻回來的，之前也翻回來過」。
→ 陷阱 #4 確認偏誤 + #12 近因偏誤。ST翻空 = 出場信號。不分析、不預測、執行。

19. 你持有5筆單，其中3筆賺1筆虧1筆平。你只想平賺的留虧的。
→ 陷阱 #1 處置效應。正確：平虧的留賺的。Cut losers, let winners run.

20. 大家（社群/新聞）都在恐慌賣出。你也想跟著賣。
→ 可能是對的（趨勢跟隨），也可能是 #9 的反面。判斷標準：ST方向。如果 ST 空 → 跟著賣是對的。如果 ST 多 → 恐慌是噪音。

21. 你上個月靠白銀賺了$30K。這個月白銀趨勢不明，但你堅持只做白銀。
→ 陷阱 #12 近因偏誤 + #11 稟賦效應（對白銀有「感情」）。正確：趨勢不明就不做。換品種或空手。

22. 你設了止損 $300，被打到了。你馬上同方向再進場（因為覺得「止損打到就會反彈」）。
→ 陷阱 #9 賭徒謬誤。止損被打 = 方向錯了。再進場同方向 = 再錯一次。等新信號。

23. 市場已經漲了兩天，你還沒進場。你焦慮：「再不進就來不及了。」
→ FOMO（Fear Of Missing Out）= #5 損失厭惡的變體。沒賺到不是虧。追高才是虧。等回調再進。

24. 你的系統（ST）說做多，但你「感覺」今天會跌。你選擇不進場。
→ 可以的。但如果你的系統長期有正期望值，持續override它會降低整體報酬。定規則：系統信號 > 感覺，除非有明確的新資訊（如CPI即將公布）。

25. 你用 $20K 帳戶做了一筆 $15K 的倉位（75% 倉位）。「這次很有把握。」
→ 陷阱 #7 過度自信 + #13 控制錯覺。沒有任何一筆交易值得用75%倉位。最大30%。

26. 你虧了很多之後，開始覺得「市場在針對我」「莊家在看我止損」。
→ 陷阱：自我中心偏誤。市場不認識你。你的止損位置恰好在很多人的止損區（因為大家用一樣的技術分析）。解法：止損設寬一點或用 ST 而非固定價格。

27. 5月你賺了$40K，覺得應該犒賞自己→放鬆了6月的紀律。
→ 陷阱 #6 熱手謬誤 + 道德許可效應（moral licensing）。「我已經夠好了，可以放鬆一下」。= 你的三週虧損的起點。紀律沒有假期。

28. 報告說「偏空觀望」但價格在上漲。你想：「報告錯了，我要做多。」
→ 如果你有自己的判斷系統且信號明確 → 可以。但如果只是「看到漲就想追」→ 陷阱 #23 FOMO。判斷標準：你的ST方向是什麼？

29. 你決定7月開始嚴格執行紀律。但「今天先不用，明天再開始。」
→ 陷阱 #15 現狀偏誤 + 拖延。「明天開始」= 永遠不開始。紀律從現在這一秒開始。

30. 你讀完這30題，覺得「我現在都懂了，不會再犯了。」
→ 陷阱 #7 過度自信。知道 ≠ 做到。這些陷阱是大腦結構性的bug，不是「知道就能避免」的。需要反覆練習+設系統性規則。下週再做一次這30題，看直覺答案是否改變。

總結：你的防火牆系統

每次交易前的 5 秒鐘檢查（背下來）

  1. 我的止損在哪？（如果答不出來 → 不做）
  2. 倉位超過 30% 嗎？（超過 → 減到 30%）
  3. 今天有重大事件嗎？（有 → 減倉或不做）
  4. 我是在報復上一筆虧損嗎？（是 → 不做）
  5. 如果這筆全虧，我明天還能交易嗎？（不能 → 倉位太大）

每次想「不止損」時的 3 秒鐘自問

  1. 如果我現在是空手，會在這個價位開這個方向嗎？（No → 平）
  2. 我不走是因為「分析」還是因為「不甘心」？（不甘心 → 平）
  3. 這筆繼續虧下去，最壞會怎樣？（如果答案讓你害怕 → 平）
