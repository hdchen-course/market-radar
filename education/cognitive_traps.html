交易認知陷阱 — 你的大腦在騙你

15 個最致命的決策偏誤 + 30 題情境訓練 — 每次虧損背後都有一個認知 Bug

為什麼需要這篇：交易者虧的錢往往不是因為「不會賺」，而是來自「大腦的系統性判斷錯誤」— 不是情緒失控，而是思考本身有 Bug。這些 Bug 是人類演化留下的，每個人都有，但可以透過覺察和訓練來規避。

致命陷阱 #1 ~ #5：持倉時的判斷扭曲

#1 處置效應（Disposition Effect）

定義：急著賣贏的、死抱虧的。賺一點就跑，虧很多才走。

原因：人對「確定的小收益」的快樂 > 「可能的大收益」的期望。但對虧損，人寧可賭一把「可能回本」也不願接受「確定的小虧損」。

常見案例：做多白銀在 $72 附近 → 跌到 $67（設好的止損線）→ 不執行（不想確認虧損）→ 最終跌到 $61 爆倉。同時之前賺的單子都是「小賺就跑」。= 典型處置效應：贏的跑太快，虧的扛太久。
解藥：
  • 開倉前就寫下：「這筆我最多虧 $___，到了就走。」
  • 把止損單掛上去，不取消。讓系統替你做決定，繞過大腦。
  • 反過來想：如果你現在是空手，會不會在這個價位開多？如果不會 → 那你為什麼還抱著？

#2 沉沒成本謬誤（Sunk Cost Fallacy）

定義：「已經虧這麼多了，不甘心現在走」— 用過去的投入來合理化繼續持有。

原因：大腦不想承認之前的決定是錯的。平倉 = 承認失敗。所以繼續抱 = 「還沒失敗」的幻覺。

常見案例：成本 $72，跌到 $67（止損線）時心想：「已經虧了 $5/oz，如果現在走就確定虧了。萬一明天漲回來呢？」→ 結果繼續跌到 $61。
正確思考：「虧了 $5 是已經發生的事實，跟未來方向無關。唯一的問題是：從 $67 開始，更可能漲還是跌？」
解藥 — 「Clean Sheet」思維：
  • 假裝你現在是空手。你會在這個價位開這個方向嗎？
  • 如果答案是 No → 立刻平倉。你的成本跟未來方向毫無關係。
  • 口訣：「成本是歷史，價格是現在，方向是未來。三者無關。」

#3 錨定效應（Anchoring）

定義：被一個「參考數字」綁住判斷。例如被自己的成本價、前高、或目標價錨定。

原因：大腦需要一個「定錨」來做判斷。一旦設定，很難擺脫。

常見模式：
  • 錨定成本：「$72 進的，至少等到 $70 才平」→ 結果 $70 到不了，一路跌到爆倉
  • 錨定目標：設目標 $70 → 到 $68.5 時「再等一下就到了」→ 結果反轉暴跌
  • 錨定前高：「上週還 $75，現在才 $67，太便宜了」→ 但市場不在乎你的「太便宜」
解藥：
  • 問自己：「如果我不知道成本/前高/目標，只看現在的圖和數據，我會做什麼？」
  • 用 SuperTrend 方向代替主觀判斷。ST 不會被錨定。
  • 把目標價視為「建議」而非「必須」。到了附近就開始減倉，不等精確命中。

#4 確認偏誤（Confirmation Bias）

定義：只看支持自己方向的資訊，忽略反面證據。

原因：持有多單時，大腦會自動過濾掉「看空」的資訊，放大「看多」的資訊。

常見案例：三框全多時 → 只看到「趨勢向上、金銀比收斂」→ 忽略了「CPI 超預期、Fed 升息、30Y 破 5%」這些巨大利空。結果：SuperTrend 在 6 小時內從三框全多 → 雙空。選擇性忽略宏觀面惡化是確認偏誤的典型表現。
解藥 — 「魔鬼代言人」練習：
  • 每次開倉後，強迫自己寫下 3 個「這筆可能錯的理由」
  • 設定「反轉信號」：不是等跌了才想，是開倉時就定義「什麼信號代表我錯了」
  • 看報告時先看風險警告，再看多頭論述。強制自己先消化負面資訊。

#5 損失厭惡（Loss Aversion）

定義：虧 $1 的痛苦是賺 $1 快樂的 2-2.5 倍。所以人會冒更大風險去「避免確認虧損」。

原因：演化設計 — 在野外，失去食物（餓死）比得到額外食物更重要。但在交易中這個本能是有害的。

常見案例：「不想讓今天變成虧損日」→ 浮虧的單子不平 → 小虧變大虧。解法：目標從「每天獲利」改為「每週獲利」— 允許虧損日存在。
解藥：
  • 重新定義「贏」：執行了止損 = 贏（贏在紀律）。虧了 $300 但守住紀律 = 成功的交易。
  • 數學化：一週5天，3天賺+2天虧，但賺的>虧的 = 週正。允許虧損日存在。
  • 口訣：「浮虧就是虧，早認早解脫，明天再賺。」
持倉陷阱核心總結：這五個陷阱的共同根源是「不願確認虧損」。大腦把「未實現虧損」當作「還沒虧」，只有平倉才算「真的虧了」。但事實是：浮虧就是虧。你帳戶裡的錢已經少了。越早面對，損失越小。

致命陷阱 #6 ~ #10：連勝後的判斷失靈

#6 熱手謬誤（Hot Hand Fallacy）

定義：連贏幾次後以為自己「手感好」「看盤很準」，開始放鬆紀律、加大倉位。

原因：大腦把隨機的連勝解讀為「我的技術變好了」。實際上市場的隨機性不會因為你之前贏了就減少。

常見案例：連續四天每天 +3% → 第五天覺得「手感好」→ 加大倉位 → 遇到 CPI 衝擊 → 爆倉。
連勝造成的三重危險：(1) 放鬆止損 (2) 加大倉位 (3) 覺得「看懂了」。三者合一 = 災難。
解藥：
  • 連贏 3 天後，第 4 天倉位減半。不是因為會虧，是因為你的判斷力在「贏」的興奮中下降了。
  • 連贏時問自己：「如果明天突然虧3天的利潤，我能接受嗎？」如果不能 → 你倉位太大了。
  • 設固定規則：無論贏多少天，單筆倉位不超過帳戶的 X%。永遠。

#7 過度自信（Overconfidence）

定義：高估自己預測市場方向的能力。「我看得很準」「我理解市場了」。

原因：賺錢時歸功於自己（技術好）。虧錢時歸咎於外部（市場不正常）。這叫「自利歸因偏誤」。

常見案例：連續幾個月獲利 → 覺得自己已經「懂了」→ 放鬆風控 → 幾週內虧回去甚至更多。事實：獲利期間標的恰好在明確趨勢中。任何人在那個趨勢中順勢操作都會賺。不全是「個人技術」。
解藥：
  • 把每次獲利歸因為：50% 市場環境 + 30% 紀律 + 20% 技術。不是 100% 自己厲害。
  • 問：「如果把我的操作反過來做，會虧多少？」如果答案是「也不會虧太多」→ 代表市場在送錢，不是你在賺錢。
  • 連贏時最危險。把紀律規則寫在螢幕旁邊，連贏時更要看。

#8 目標漂移（Goal Creep）

定義：原本設好目標，到了之後改成更高的目標。「已經到 $68 了，再等等看能不能到 $70」。

原因：當目標接近時，大腦開始計算「如果繼續走會賺更多」。放棄「確定的利潤」讓人痛苦。

常見模式（反覆出現）：
  • 持有加密貨幣有賺沒跑 → 改更高目標 → 最後大虧
  • 白銀接近目標價（差 $1-2）→ 「再等一下」→ 反轉暴跌
  • 重大事件後有利潤 → 沒平 → 方向反轉
多數虧損都有「到了沒跑」的影子。
解藥：
  • 到了就走。不改目標。永遠。這是最需要內化的一條規則。
  • 到目標先平一半（鎖利潤）。剩一半設 trailing stop。這樣「繼續漲」你也有，「反轉」你也不虧。
  • 開倉時把目標寫在紙上。到了的時候不看盤，直接按照紙上寫的執行。

#9 賭徒謬誤 / 均值回歸幻覺

定義：「已經連跌5天了，今天應該要漲了吧？」— 把隨機事件當作有「欠債」要還。

原因：大腦不理解「趨勢」。在趨勢中，連跌 5 天之後連跌第 6 天的機率跟前5天無關。

常見案例：白銀從 $75 連跌到 $65 → 「跌這麼多了應該反彈了吧」→ 加倉 → 繼續跌到 $61。趨勢不會因為「已經跌很多」就停止。
解藥：
  • 不預測反轉。只在反轉確認後跟隨。（1h/4h ST翻轉 = 確認）
  • 「連跌很多」不是做多理由。「ST翻多」才是做多理由。
  • 口訣：「趨勢的持續時間永遠比你以為的長。」

#10 資訊過載 → 分析癱瘓 or 過度交易

定義：看太多資訊反而無法做決定，或者每個資訊都想反應（頻繁進出）。

原因：現代交易者有無限資訊。每一條新聞都感覺「重要」。大腦無法區分噪音和信號。

解藥：
  • 定義你的「決策輸入」只有3個：SuperTrend方向 + 多空計分 + 重大事件日曆。其他都是噪音。
  • 開盤前做一次分析就好。盤中不再重新分析（除非觸發止損）。
  • 後段小單磨的階段，不看新聞。只看價格和掛單成交。
連勝陷阱核心總結：連勝最危險的不是「下一筆會虧」，而是連勝改變了你的行為模式：倉位變大、止損變鬆、標準變低。這三個改變加起來，讓「一次普通的虧損」變成「毀滅性的打擊」。

致命陷阱 #11 ~ #15：出場與風控的判斷錯誤

#11 稟賦效應（Endowment Effect）

定義：你擁有的東西比實際更值錢。持有的倉位比實際更「捨不得」。

常見案例：白銀多單持有多天 → 開始對它有「感情」→ 變成「我的白銀」→ 更不願意平掉。
解藥：倉位只是數字。不是「你的」。它不認識你。每天問：「如果我現在是空手，我會不會在這個價格建立這個倉位？」

#12 近因偏誤（Recency Bias）

定義：過度看重最近發生的事。「昨天反轉搶單賺了 → 今天再搶一次」。

常見案例：「反轉搶單第一次有效，第二次出問題」— 第一次成功讓大腦覺得「這招有效」，忽略了第二次環境已不同。
解藥：看長期統計，不看最近幾次。「反轉搶單」的長期勝率可能只有40% — 第一次贏只是運氣。規則：每天只做一次。

#13 控制錯覺（Illusion of Control）

定義：以為自己能控制結果。「我看著盤就不會虧」「我手動比止損單好」。

常見案例：不設止損單（「我會自己手動停損」）→ 凌晨睡著 → 醒來已經崩了。你覺得你能控制，但你控制不了自己會睡著。
解藥：
  • 永遠掛止損單。不依靠「我會手動停損」。你會睡著、會猶豫、會改主意。系統不會。
  • 承認：你無法控制市場方向。你能控制的只有：倉位大小、止損位置、進場時機。

#14 結果偏誤（Outcome Bias）

定義：用結果來判斷決策品質。「攤平最後賺了 → 所以攤平是對的」。

常見案例：「上次攤平也成功了」— 下次攤平時大腦會用這個記憶來合理化錯誤操作。
解藥：判斷決策品質要看「過程」不是「結果」。
  • 好決策+壞結果：設了止損被打到（然後市場反轉了）→ 仍然是好決策。
  • 壞決策+好結果：攤平扛單最後回本 → 仍然是壞決策（下次可能爆倉）。
  • 只獎勵自己「好過程」，不獎勵「好結果」。

#15 現狀偏誤（Status Quo Bias）

定義：維持現狀感覺比改變安全。「已經持有了，就繼續持有吧」— 即使該平倉。

常見案例：白銀浮虧 → 「維持現狀（不動）比做一個動作（平倉）容易」→ 不動 = 繼續持有 = 虧更多。不做決定本身就是一個決定（決定繼續承擔風險）。
解藥：把「不動」也視為一個主動選擇。每天問：「我今天是 選擇 繼續持有，還是 懶得 做決定？」如果是後者 → 立刻行動。
出場陷阱核心總結：出場比進場難十倍，因為出場意味著「確認結果」。大腦會用各種方式迴避這個確認：對倉位有感情（#11）、用過去合理化現在（#12）、覺得自己能控制（#13）、用好運合理化壞策略（#14）、不想做任何改變（#15）。解法都一樣：用系統規則代替人工判斷。

情境 1

你白銀多單成本 $72，現在跌到 $67。你設的止損是 $67。價格剛碰到 $67，你在想：「$67 好像是支撐位，也許會彈回來，我先取消止損看看。」你應該怎麼做？

  • A. 取消止損，觀察支撐位是否有效
  • B. 讓止損執行，不取消
  • C. 取消止損並加倉攤平成本
  • D. 把止損下移到 $65 給更多空間
正確：B. 讓止損執行，不取消。
觸發陷阱：#1 處置效應（不想確認虧損）+ #5 損失厭惡 + #13 控制錯覺
「也許會彈回來」= 確認偏誤（#4），只看到支持自己想法的理由。
規則：止損觸發就走。不思考。不分析。走。

情境 2

你連續三天每天賺 3%。今天第四天，你想：「手感好，今天可以加大倉位試試看。」正確做法是？

  • A. 加大倉位，趁勝追擊
  • B. 維持倉位不變
  • C. 維持原本倉位甚至減半
  • D. 全倉操作，把握連勝勢頭
正確：C. 維持原本倉位甚至減半。
觸發陷阱：#6 熱手謬誤 + #7 過度自信
連贏三天不代表第四天也會贏。「手感」不是真的。市場不會因為你前三天贏就今天也讓你贏。
規則：連贏3天 → 第4天倉位不變或減半。

情境 3

白銀到了 $69.5，目標是 $70。心想：「這麼接近了，不差 $0.5，等到 $70 再說。」正確做法是？

  • A. $69.5 就平一半，不等精確命中 $70
  • B. 等到 $70 精確命中再全部出場
  • C. 把目標改成 $72，趨勢還在
  • D. 設 $70 限價單等待成交
正確：A. $69.5 就平一半。
觸發陷阱：#3 錨定效應（被 $70 目標錨定）+ #8 目標漂移
等精確命中 → 常常差一點就反轉。然後又被新價格錨定（「等回到 $69.5」）。
規則：目標附近（±2%）就開始減倉。不等精確命中。

情境 4

CPI 今晚公布。你持有多單浮盈 +2%。你想：「如果 CPI 低，可能再漲 3%。賭一把。」正確做法是？

  • A. 全部持有，賭 CPI 結果
  • B. CPI 前至少平一半，鎖利潤
  • C. 加倉，因為有信心 CPI 會低
  • D. 全部平倉，完全避險
正確：B. CPI 前至少平一半，鎖利潤。
觸發陷阱：#7 過度自信（以為能預測CPI結果）+ #8 目標漂移（+2%不夠，想要+5%）
CPI 結果是 50/50 賭博。你有 +2% 的確定利潤 vs「可能+5%也可能-5%」的賭博。
規則：重大事件前鎖利潤。不賭。已經有的利潤 > 可能的利潤。

情境 5

反轉搶單第一次賺了 $200。市場又到了一個你覺得「過度」的位置。你想再搶一次反轉。正確做法是？

  • A. 再搶一次，剛才的分析邏輯一樣適用
  • B. 不做第二次，收工或改用其他模式
  • C. 用更大倉位搶，因為第一次驗證了方向
  • D. 等跌更多再搶，加大勝率
正確：B. 不做第二次。收工或改用其他模式。
觸發陷阱：#6 熱手謬誤 + #12 近因偏誤（剛才成功 → 以為下次也會）
反轉搶單只有第一次準。第二次通常出問題。
規則：反轉搶單每天只做一次。做完收工。

情境 6

你今天早上虧了 $400。現在看到一個「很好的機會」，想把虧損賺回來。正確做法是？

  • A. 進場，這個機會確實很好
  • B. 用加大倉位進場，一筆賺回來
  • C. 用小倉位試試看
  • D. 今天不做了，明天再來
正確：D. 今天不做了。明天再來。
觸發陷阱：#5 損失厭惡（急於把虧損「修復」）+ #7 過度自信
虧損後的「好機會」通常是大腦在急於報復（revenge trading）。判斷力在虧損後是降低的。
規則：日初虧損 → 當天停止交易。

情境 7

白銀已經從 $75 跌到 $65（-13%）。你想：「跌這麼多了，肯定快反彈了，我來接。」正確做法是？

  • A. 進場做多，跌幅已經夠大

你的防火牆系統

核心理念：不靠意志力對抗認知偏誤 — 用系統規則繞過大腦。把下面的清單背下來，在每次交易的關鍵時刻啟動。

每次交易前的 5 秒鐘檢查（背下來）

  1. 我的止損在哪？（如果答不出來 → 不做）
  2. 倉位超過 30% 嗎？（超過 → 減到 30%）
  3. 今天有重大事件嗎？（有 → 減倉或不做）
  4. 我是在報復上一筆虧損嗎？（是 → 不做）
  5. 如果這筆全虧，我明天還能交易嗎？（不能 → 倉位太大）

每次想「不止損」時的 3 秒鐘自問

  1. 如果我現在是空手，會在這個價位開這個方向嗎？（No → 平）
  2. 我不走是因為「分析」還是因為「不甘心」？（不甘心 → 平）
  3. 這筆繼續虧下去，最壞會怎樣？（如果答案讓你害怕 → 平）

連贏後的自我降溫清單

  1. 連贏 3 天 → 第 4 天倉位減半
  2. 連贏 5 天 → 休息一天不做
  3. 月報酬 > 15% → 剩餘交易日倉位減至 20%
  4. 問自己：如果明天虧 3 天利潤，能接受嗎？（不能 → 倉位太大）

虧損後的行為規範

  1. 單筆虧損 > 帳戶 3% → 當天停止交易
  2. 連虧 2 筆 → 當天停止交易
  3. 虧損後想加大倉位 → 這是 revenge trading → 縮小倉位
  4. 虧損後看到「好機會」→ 90% 是大腦在騙你 → 不做

重大事件日規範

  1. CPI / FOMC / NFP 前 → 至少平一半倉位
  2. 有浮盈 → 設 trailing stop 保護利潤
  3. 不在事件前 30 分鐘開新倉
  4. 事件後等 15 分鐘波動消化再決策
