指標使用限制

鏈上指標 提供長期趨勢判斷，不適合短線進出場 timing。NUPL/MVRV 適合判斷週期位置，不適合日內交易。
數據延遲：鏈上數據通常有 1-24 小時延遲，不是即時交易信號。
生態變化：ETF、L2、跨鏈橋等新基礎設施會改變傳統鏈上指標的有效性。

鏈上基礎

▶

什麼是鏈上數據？

核心概念

鏈上數據 (On-Chain Data) 是直接從區塊鏈上提取的交易和活動資訊。由於區塊鏈是公開帳本，所有交易都是透明且不可篡改的 — 這讓我們可以追蹤「大戶行為」。

為什麼鏈上數據有效？
  • 透明性：每筆交易永久記錄在區塊鏈上，任何人都可以驗證
  • 不可偽造：不像社交媒體情緒可以被操縱，鏈上行為需要真金白銀
  • 先行指標：大戶的鏈上行為（轉帳、質押、提幣）往往先於價格變動
  • 全面性：涵蓋所有參與者 — 從散戶到鯨魚到礦工

鏈上 vs 鏈下數據

類型來源範例特性
鏈上 (On-Chain)區塊鏈公開帳本UTXO 移動、智能合約互動、Gas 費透明、不可篡改、有延遲
鏈下 (Off-Chain)交易所、社交媒體Order book、Funding Rate、Twitter 情緒即時但可被操縱
混合型交易所 + 鏈上Exchange Flow（交易所流入/流出）結合兩者優勢

UTXO 模型 vs Account 模型

  • UTXO (Bitcoin)：每筆交易消耗舊的輸出、產生新的輸出。可以精確追蹤每個「幣」的歷史，計算持有時間和成本
  • Account (Ethereum)：類似銀行帳戶模型。追蹤地址餘額變化，但無法像 UTXO 一樣精確追蹤「哪顆幣」
  • 影響：BTC 的鏈上指標（如 SOPR、Coin Days Destroyed）基於 UTXO，比 ETH 更精確

鏈上數據的核心分析框架

四大分析維度

維度問什麼代表指標
估值市場被高估還是低估？NUPL, MVRV, NVT Ratio
行為參與者在做什麼？SOPR, Exchange Flow, Active Addresses

分析原則：
  • 單一指標永遠不足以做決策 — 至少看 3-4 個不同維度的指標
  • 鏈上指標是「方向指南針」而非「精確計時器」— 適合判斷週期位置，不適合抓精確頂底
  • 結合宏觀環境（利率、DXY）使用效果更好

BTC 鏈上指標

▶

NUPL (Net Unrealized Profit/Loss)

定義

NUPL = (Market Cap - Realized Cap) / Market Cap

衡量整個市場參與者的「帳面盈虧」狀態。Realized Cap 是每個 UTXO 最後一次移動時的價格加總。

實戰規則

NUPL 區間狀態操作建議
> 0.75Euphoria分批出場 歷史上都是頂部
0.5 - 0.75Greed開始減倉、提高警覺
0.25 - 0.5Optimism持有，趨勢仍在
0 - 0.25Hope開始建倉
< 0Capitulation極端抄底機會
▶

MVRV (Market Value to Realized Value)

定義

MVRV = Market Cap / Realized Cap

衡量市場目前的定價相對於所有持有者的平均成本（Realized Cap）是高估還是低估。

關鍵閾值

MVRV含義歷史對應
> 3.5極度過熱，所有人平均獲利 250%+2017/12, 2021/4
2.0 - 3.5偏高但未到極端牛市中段
1.0 - 2.0正常範圍大部分時間
< 1.0所有人平均虧損！2018/12, 2020/3, 2022/11

MVRV Z-Score

Z-Score = (Market Cap - Realized Cap) / Std(Market Cap)

Z-Score 更精確：消除了 market cap 成長帶來的偏差。Z-Score > 7 = 賣出，Z-Score < 0.1 = 買入。

MVRV 的變體

  • STH-MVRV：只看短期持有者的 MVRV — 更敏感，適合判斷短期支撐/阻力
  • LTH-MVRV：只看長期持有者的 MVRV — 更穩定，適合判斷大週期
  • 實戰：STH-MVRV < 1 + LTH-MVRV > 1 = 短期痛苦但長期持有者仍獲利 = 修正非熊市
▶

STH / LTH 成本線 + Supply 分佈

定義

  • STH (Short-Term Holder)：持有 < 155 天的地址
  • LTH (Long-Term Holder)：持有 > 155 天的地址
  • Realized Price：該群體所有 UTXO 的加權平均成本

解讀框架

價格位置含義歷史出現
Price > STH Cost短期持有者獲利 → 市場健康正常牛市
Price < STH Cost短期持有者虧損 → 恐慌拋售加速修正初期
Price < LTH Cost長期持有者也虧損 → 極端熊市底部2015, 2018/12, 2022/11
牛市中的支撐：在牛市中，BTC 價格回測 STH Realized Price 通常是絕佳買點（類似傳統技術分析中的「回測均線」）。

STH-SOPR 組合

當 STH 的 SOPR < 1（短期持有者在虧損狀態下賣出）+ 價格觸及 STH Realized Price = 高概率反轉點。

LTH/STH Supply Ratio

  • LTH Supply 佔比上升：更多 BTC 被長期鎖定 = 流動供給減少 = 長期利好
  • LTH Supply 佔比下降：長期持有者開始分發 = Distribution 階段開始
  • 歷史規律：每個牛市頂部前 3-6 個月，LTH Supply 開始下降
  • Illiquid Supply：從不移動或極少移動的 BTC 數量 — 持續上升代表「鑽石手」增加
▶

SOPR (Spent Output Profit Ratio)

定義

SOPR = 賣出時價值 / 買入時價值（所有當天移動的 UTXO 加總）

SOPR 值含義信號
> 1平均而言賣出者在獲利牛市中回測 1.0 是支撐
= 1盈虧平衡關鍵支撐/阻力
< 1平均而言賣出者在虧損投降信號 / 底部信號
牛市法則：在確認的牛市中，SOPR 回測 1.0 並反彈 = 買入機會（人們在成本價選擇不賣）。
熊市法則：在熊市中，SOPR 反彈到 1.0 後再度下跌 = 做空機會（人們在回本時急忙出逃）。

aSOPR (Adjusted SOPR)

移除了 coin age < 1 hour 的交易（排除交易所內部轉帳和 change outputs），更能反映真實的獲利/虧損行為。

SOPR 的進階用法

SOPR 變體定義最佳用途
STH-SOPR只看短期持有者的 SOPR判斷短期市場情緒和支撐/阻力
LTH-SOPR只看長期持有者的 SOPR判斷大週期的分發/積累
aSOPR去除 noise 的調整版最乾淨的通用信號
Entity-Adjusted SOPR以實體（非地址）為單位消除同一人多地址的干擾
LTH-SOPR 進場的週期信號：
  • LTH-SOPR 從 > 1 持續下降到接近 1 = LTH 獲利空間壓縮 = 分發減速
  • LTH-SOPR < 1 = 長期持有者在虧損賣出 = 極端投降 = 歷史底部
  • LTH-SOPR 從 < 1 回升突破 1 = 投降結束，新一輪上漲開始
▶

Exchange Flow + 礦工指標

交易所淨流出 / 流入

Exchange Netflow = 流入交易所的 BTC - 流出交易所的 BTC

情境Netflow含義方向
大量流入> 0準備在交易所賣出Bearish
大量流出< 0提幣到冷錢包 = 長期持有Bullish
鯨魚單筆大額流入> 0大戶即將出貨Strong Bearish
注意事項：
  • 需要區分是「真正的個人錢包流出」還是「交易所之間的內部轉帳」
  • ETF 上市後，部分流入可能是 ETF custodian 的調度，不是散戶賣出
  • 最好結合「鯨魚地址追蹤」一起看

礦工指標 + 減半週期

指標定義信號
Hash Ribbon30D vs 60D Hash Rate MA30D < 60D = 礦工投降 = 底部信號
Puell MultipleDaily Miner Revenue / 365D MA Revenue< 0.5 = 低估, > 4 = 過熱
Miner Outflow從礦池地址流出到交易所的 BTC大額流出 = 礦工需要套現 = 賣壓
Difficulty Ribbon不同 MA 期間的挖礦難度帶壓縮 = 礦工投降結束 = 買入

減半週期效應

  • 減半前 6 個月：市場開始 price in，通常溫和上漲
  • 減半後 3-6 個月：供給衝擊開始顯現，加速上漲
  • 減半後 12-18 個月：歷史上每次都達到新 ATH
  • 注意：每次減半效應遞減（區塊獎勵佔比越來越小）
Stock-to-Flow 模型：雖然已被部分「證偽」（2022 年嚴重偏離），但其核心邏輯（供給減少 + 需求增加 = 價格上漲）仍然有效。不要作為精確定價模型使用，但供給衝擊的方向性判斷仍有參考價值。

ETH + DeFi 鏈上指標

▶

ETH Staking + Gas + Burn

1. Staking Rate（質押比例）

指標BullishBearish
Staking Ratio 上升供給減少 + 長期信心增加-
Staking Ratio 下降-validators 退出 = 信心動搖
Staking Queue 增長需求旺盛-
Exit Queue 增長-大規模退出 = 風險

2. Gas Fee（Gas 費用）

  • High Gas (> 50 gwei)：鏈上活躍度高 = 使用需求強 = 生態健康
  • Low Gas (< 10 gwei)：鏈上沉寂 = 需求疲弱 = 可能見底也可能持續低迷
  • Gas Spike：通常由 NFT mint、DeFi 清算、或新協議上線引發

3. ETH Burn Rate (EIP-1559)

EIP-1559 機制：每筆交易的 base fee 被永久銷毀。
  • High Burn：大量 ETH 被銷毀 → 供給通縮 → 長期利好價格
  • Low Burn：銷毀量低於 PoS 新發行量 → ETH 供給通膨
  • 淨通縮日：Burn > Issuance = 每天 ETH 總量在減少

ETH/BTC Ratio 作為風險偏好指標

  • ETH/BTC 上升：市場 risk-on，資金從 BTC 流向更高 beta 資產
  • ETH/BTC 下降：市場 risk-off，資金回流 BTC 避險
  • 歷史規律：牛市後期 ETH outperform BTC → ALT season 信號

DeFi TVL 與生態健康

定義

TVL (Total Value Locked) = 所有 DeFi 協議中鎖定的資產總值。代表市場願意承擔智能合約風險的資金量。

TVL 變化含義延伸
TVL 增長生態健康，資金在鏈上工作利好 ETH（gas demand）和 DeFi tokens
TVL 暴跌風險規避，資金撤離可能有系統性風險（如 Terra/LUNA、FTX）
TVL 穩定但 Token 價格跌只是投機降溫，基本面仍在反而是好的買入時機
注意 TVL 的幻覺：TVL 以 USD 計算時，token 價格下跌會自動降低 TVL。需要看「以 ETH 計價」的 TVL 來判斷真實資金流動。

關鍵 DeFi 指標

  • DEX Volume / CEX Volume：比率上升 = 鏈上活躍度增加
  • Lending Protocol Utilization：高借款利率 = 槓桿需求高 = 牛市特徵
  • Liquidation Volume：大量清算 = 去槓桿 = 可能接近底部

DeFi 系統性風險指標

警示信號含義歷史案例
Stablecoin de-peg (> 1%)市場對 stablecoin 信心動搖UST (2022/5), USDC (2023/3 SVB)
Protocol TVL 24h -30%+可能有 exploit 或 bank runTerra/Anchor, FTX/Alameda
Lending 清算 cascade槓桿過度 → 連鎖清算2021/5 BTC 從 $58K 到 $30K
Bridge exploit跨鏈橋被駭 → 信心崩塌Ronin, Wormhole, Nomad
DeFi 風險管理原則：
  • 不在單一協議部署超過總資產 20%
  • 監控 stablecoin peg — 任何 de-peg 立即撤資
  • 留意「不可能的高利率」— 如果 yield > 20% 且來源不明 = Ponzi 結構風險
  • 使用 DefiSafety 或 DeFiLlama Risk 評分篩選協議

L2 生態與跨鏈數據

L2 對鏈上分析的影響

隨著 Arbitrum、Optimism、Base 等 L2 成長，傳統的 L1 指標可能低估了實際的鏈上活躍度。

指標L1 Only 視角L1 + L2 完整視角
Gas Fee 低「需求疲弱」可能只是活動遷移到 L2
ETH Burn 低「通膨」L2 batch posting 使用更有效率
Active Addresses 減少「用戶流失」用戶可能在 L2 上活躍
完整分析應包含：
  • L2 的 TVL 趨勢（DefiLlama L2 section）
  • L2 的日活地址和交易數
  • L2 向 L1 發布的 batch 數據量
  • 跨鏈橋的資金流向

跨鏈資金流追蹤

  • 資金流入新鏈：代表該生態有吸引力（新 dApp、空投預期）
  • 資金從新鏈流回 ETH：該生態降溫，資金回歸安全資產
  • 追蹤工具：DefiLlama Bridges、Dune cross-chain dashboards

Stablecoin 流動性

▶

Stablecoin 供給 — 加密市場的「場外流動性」

核心邏輯

Stablecoin Market Cap = 加密市場的「彈藥庫」。更多 stablecoin 被鑄造 = 更多法幣進入加密生態 = 潛在買壓。
指標BullishBearish
USDT Market Cap 增加新錢入場，尤其是亞洲/新興市場-
USDT Market Cap 減少-資金離場
USDC Market Cap 增加美國機構/合規資金入場-
Stablecoin / Total Crypto MC 比例高大量 dry powder 等待部署-

USDC vs USDT 的差異

  • USDT：全球性，亞洲主導，零售和場外交易為主
  • USDC：美國合規，機構偏好，代表「聰明錢」入場
  • 當 USDC 供給增長但 USDT 持平 = 機構在建倉

Stablecoin Dominance

Stablecoin Dominance = Stablecoin MC / Total Crypto MC

  • Dominance 高（> 10%）= 資金在觀望，尚未入場 = 未來有買壓空間
  • Dominance 低（< 5%）= 資金已 fully deployed = 缺乏增量資金

USDT/USDC Mint-Burn 作為先行指標

Mint (鑄造) = 新供給 = 新資金入場

Mint 邏輯：Tether/Circle 只會在收到法幣後才鑄造新的 stablecoin。所以每一次大額 mint = 有人存入了等量的美元。
  • $500M+ USDT Mint：大型場外交易或機構入場
  • 連續多日 Mint：持續性的資金流入 = 強烈看多
  • Mint 到交易所地址：準備直接買幣
  • Mint 到 DeFi 地址：準備部署收益策略

Burn (銷毀) = 供給減少 = 資金離場

Burn 邏輯：用戶將 stablecoin 贖回為法幣時，stablecoin 被銷毀。
  • 大額 USDT Burn：資金從加密市場撤離
  • 連續 Burn：持續性的資金流出 = 看空
  • Burn 後 Supply 明顯下降：需要觀察是「恐慌性出逃」還是「正常贖回」

交易所 Stablecoin 流入 + Exchange Stablecoin Ratio

方向判斷

Flow 方向含義信號
大量 USDT 流入交易所準備買幣Bullish
大量 USDT 從交易所流出OTC 出金 或 DeFi 部署Bearish 或 中性
Stablecoin 流入 + BTC 流出資金在「換車」：用穩定幣買 BTC 並提幣Strong Bullish

Exchange Stablecoin Ratio (ESR)

ESR = Exchange Stablecoin Balance / BTC Exchange Balance (in USD)

ESR 的含義：
  • ESR 高：交易所上穩定幣相對 BTC 很多 = 大量「子彈」準備部署 = 買壓潛力大
  • ESR 低：穩定幣已被部署（換成 BTC 了）= 缺乏增量資金
  • ESR 突然上升：大量 stablecoin 湧入但 BTC 未動 = 即將有大動作

On-Chain vs CEX Flow 的差異

數據源代表限制
On-Chain Stablecoin Flow實際的區塊鏈轉帳無法區分「買幣意圖」vs「做 DeFi」
CEX Order Book交易所內的買賣意圖可以被 spoof（假單）
CombinedStablecoin 流入 CEX = 買幣意圖明確最佳組合信號

Stablecoin De-Peg 風險監控

為什麼 De-Peg 是最嚴重的風險信號

穩定幣是加密生態的基礎設施。當穩定幣脫錨，整個生態的信任基礎動搖。

De-Peg 幅度嚴重程度應對
< 0.5%正常波動觀察，不需反應
0.5% - 2%值得關注減少該 stablecoin 敞口

衍生品鏈上指標 — Funding + OI + 清算

永續合約 Funding Rate

Funding Rate = 多頭支付給空頭（正值）或空頭支付給多頭（負值）

Funding Rate含義交易信號
> 0.05% (per 8h)多頭過度擁擠，槓桿過高過熱，可能大回調
0.01% - 0.05%正常看多共識中性偏多
0% - 0.01%平衡無信號
< 0%空頭過度擁擠Short Squeeze 風險
< -0.03%極度看空共識逆向看多

Open Interest (OI) + Price 組合判讀

OI 變化價格變化含義
OI 增加Price 上漲新多頭建倉 → 趨勢健康但注意過熱
OI 增加Price 下跌新空頭建倉 → 看空趨勢加強
OI 減少Price 上漲空頭平倉（Short Squeeze）→ 反彈可能不持久
OI 減少Price 下跌多頭投降平倉 → 去槓桿，接近底部

清算數據

清算 (Liquidation) 作為情緒指標：
  • 大量多頭清算：市場下跌引發連鎖停損 → 可能是瀑布式下跌的加速器
  • 大量空頭清算：上漲引發空頭止損 → Short Squeeze 放大漲幅
  • 清算結束後：槓桿被清除 = 市場更健康 = 可能是反轉起點
  • $1B+ 清算日：歷史上通常是極端事件的結束點，可以開始逆向布局

Funding + OI 組合策略

  • Funding 極高 + OI 新高 + Price 新高：三重過熱 → 強烈的回調信號
  • Funding 極負 + OI 增加 + Price 下跌：空頭積極建倉 → Short Squeeze 即將到來
  • OI 崩塌 + Funding 歸零：去槓桿完成 → 開始觀察新的方向信號

鏈上指標的局限性與進化

ETF 時代對鏈上分析的影響

2024 年 BTC ETF 上市後，傳統鏈上指標面臨新的挑戰：

變化對傳統指標的影響調整方式
ETF 持有 BTC 在 custodianExchange Balance 可能失真區分 ETF custodian vs 散戶交易所
ETF 資金流入/流出不經過傳統鏈上路徑需要結合 ETF Flow 數據
BTC 「紙上」交易增加鏈上活躍度可能與實際需求脫節Active Addresses 需要重新校準
機構參與度增加傳統散戶行為模式可能不再主導需要追蹤 Coinbase Premium Index
新時代補充指標：
  • ETF Net Flow：每日 ETF 淨流入/流出（替代部分 Exchange Flow 功能）
  • Coinbase Premium：Coinbase 價格 vs Binance 價格的溢價 = 美國機構需求
  • Grayscale Discount/Premium：GBTC 折溢價反映機構情緒
  • CME Futures Basis：CME 期貨溢價 = 機構的 carry trade 需求

指標失效的情境

鏈上指標可能失效的場景：
  • 黑天鵝事件：FTX 級別的交易所爆雷 — 所有指標無法預測
  • 監管突襲：政府突然禁令 — 基本面指標無法反映政策風險
  • 技術故障：協議 bug 或 hack — 鏈上數據可能出現異常讀數
  • 市場結構變化：新工具（如 ETF）改變資金路徑 — 舊指標需要重新校準

互動測驗

測試你對鏈上數據分析的理解。共 15 題。

Q1. NUPL 值為 0.82，市場最可能處於什麼階段？

NUPL > 0.75 = Euphoria。歷史上每次進入此區間都是牛市頂部，應該開始分批減倉。2017 和 2021 的頂部都出現在此區間。

Q2. 交易所出現大量 BTC 淨流入（Netflow > 0），這代表什麼？

BTC 流入交易所的主要原因是準備賣出。BTC 存入交易所後可以快速掛賣單，這是 bearish 信號。但需要排除 ETF custodian 的調度。

Q3. Stablecoin 供給增加 + 大量流入交易所 + BTC Exchange 餘額下降，綜合判斷？

Stablecoin 增加 = 新錢入場。流入交易所 = 準備買幣。BTC 餘額下降 = BTC 被買走並提到冷錢包。三者結合 = 強烈的買入信號，代表「聰明錢」在積累。

Q4. MVRV 為 0.85，SOPR 為 0.88，交易所 BTC 出現大額流入。綜合判斷？

MVRV < 1 和 SOPR < 1 都顯示市場在底部區域（所有人虧損）。但大額流入交易所 = 仍有拋壓。結合看 = 投降階段進行中，接近但可能未到最低點。分批建倉合適，不宜一次梭哈。

Q5. 在牛市中，BTC 的 SOPR 回測 1.0 後反彈，應該如何操作？

牛市中 SOPR 回測 1.0 = 賣出者「回本就不賣了」= 支撐確立。這是經典的牛市中加倉機會，歷史上多次應驗（2017, 2020-2021 多次回測）。

Q6. ETH Gas Fee 持續低迷（< 5 gwei）配合 ETH Burn Rate 低於每日發行量，代表什麼？

低 Gas = 鏈上需求不足。Burn < Issuance = ETH 淨供給在增加（通膨）。兩者結合 = 對 ETH 的價格有雙重負面影響。雖然 L2 遷移可以解釋部分原因，但對 ETH L1 的價值捕獲確實是利空。

Q7. USDC 供給快速增長但 USDT 持平，這代表什麼？

USDC 代表美國合規資金和機構。USDC 獨立增長 = 機構在主動增加加密市場敞口。這通常預示著更大的資金流入，是 bullish 信號。USDT 代表亞洲/散戶；只有 USDC 漲 = 機構主導。

Q8. DeFi TVL 以 ETH 計價仍在增長，但以 USD 計價在下跌。代表什麼？

以 ETH 計價 TVL 增長 = 實際上更多 ETH 被部署到 DeFi 中 = 使用需求真實增長。USD 計價下跌只是因為 ETH 價格跌了。這反而可能是好的買入時機 —— 基本面在改善但價格被壓制。

Q9. 以下哪個組合最代表 Wyckoff Distribution 階段？

Distribution = 頂部出貨。特徵：NUPL 極高（所有人獲利 = 出貨動機強）、Exchange Inflow（準備賣）、LTH 分發、高 Funding（槓桿多頭過熱）。A = Accumulation, B = Markup, C = Markdown。

Q10. BTC 價格跌破 STH Realized Price，但 LTH 未賣出。判斷？

Price < STH Cost = 短期持有者虧損 → 恐慌拋售可能加速（短期利空）。但 LTH 不賣 = 長期持有者仍有信心 = 不是系統性崩盤。通常是修正而非熊市底部。歷史上真正的底部需要 LTH 也投降（Price < LTH Cost）。

Q11. Funding Rate 為 -0.03%，同時 Open Interest 在增加，代表什麼？

負 Funding = 空頭佔多數（空頭付錢給多頭）。OI 增加 = 新倉在建立。結合 = 空頭在積極建倉。但這也意味著如果價格上漲，大量空頭被迫平倉 = short squeeze 風險高。

Q12. 大額 USDT Mint ($1B) 出現，但 24 小時內未流入任何交易所。最可能的解讀？

大額 Mint 代表有人存入了真實的美元換取 USDT。未立即流入交易所可能是：(1) 場外 OTC 大宗交易，(2) 機構在準備分批建倉。無論哪種，新的 $1B 進入生態 = 長期 bullish。通常 Mint 後 1-3 週會看到市場影響。

Q13. Curve 3pool 中 USDT 比例從 33% 上升到 55%。代表什麼？

Curve 3pool 的平衡反映市場對各 stablecoin 的信心。USDT 比例上升 = 人們在拋售 USDT 換成其他 stablecoin = 市場對 USDT 信心下降。這是 de-peg 風險的早期信號，需要密切關注。2022 年 Luna 崩盤時也出現過類似情況。

Q14. Hash Ribbon 發出買入信號（30D MA 重新超越 60D MA），同時 MVRV < 1。操作建議？

Hash Ribbon 買入信號 = 礦工投降結束（最弱的礦工已退出，壓力釋放）。配合 MVRV < 1 = 估值在歷史低位。兩者組合在 2015, 2019, 2020, 2022 底部都出現過，是最可靠的底部信號之一。開始建倉（分批）是正確做法。

Q15. 完整面板：NUPL 0.68 | LTH Supply 開始下降 | Stablecoin Mint 停滯 | Funding 0.06% | Exchange Inflow 增加。你的操作？

NUPL 0.68（接近 Greed/Euphoria）+ LTH 開始賣 + Stablecoin 不再增長 + 高 Funding + Exchange Inflow = 4-5 個 Distribution 信號。雖然尚未確認頂部（NUPL 沒到 0.75），但風險明顯增加。正確做法是減倉保護利潤 + 設停損，而非追高。
