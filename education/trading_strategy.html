交易策略與日常作業流程

系統化交易 SOP — 從台灣時區出發的美股/加密/期貨交易框架

⚠️ 策略驗證

本頁策略框架基於交易實務慣例而非嚴格回測。任何策略在實盤使用前應先進行：(1) 模擬盤驗證至少 30 筆 (2) 計算期望值 = 勝率 × 平均盈利 - 敗率 × 平均虧損 (3) 確認正期望值後再投入真金白銀。
注意 Overfitting 風險：針對過去數據完美優化的策略通常在實盤中失效。策略越簡單、參數越少，越可能在未來有效。

交易日程與時區

身在台灣（UTC+8），你的交易節奏必須配合美股、歐洲盤和加密市場的時間差。以下是完整的時段對照與策略建議。

核心原則：計劃交易，交易計劃 — 沒有計劃 = 不交易。每個時段都有特定的任務，不要在觀察時段下單。

行前準備 SOP

每日開盤前必做的準備清單，建議在 20:00-21:30 執行。總計約 27 分鐘。

核心原則：保護本金第一，獲利第二 — 準備工作的目的是確保你帶著清晰的頭腦和明確的計劃進入市場，而不是盲目開槍。
▶

1. 宏觀快速掃描（5 分鐘）

必做

檢查清單：

  • DXY（美元指數）今日方向 — 美元走強通常壓制風險資產
  • 30Y / 10Y 殖利率變化 — 殖利率飆升 = 成長股壓力
  • VIX 水平 — <15 低波動 / 15-20 正常 / 20-25 警戒 / >25 高波動（縮小倉位）
  • 今日重要數據/事件 — CPI/PPI/NFP/FOMC/財報
  • 隔夜主要變化 — 亞歐盤有無重大消息
快速判斷框架：
DXY 弱 + 殖利率降 + VIX 低 = Risk-On（做多科技/加密）
DXY 強 + 殖利率升 + VIX 升 = Risk-Off（做空或觀望）
混合信號 = 小倉位或不交易
▶

2. 技術面準備（10 分鐘）

必做

逐項檢查：

  • 各持倉資產 SuperTrend 三時框狀態
    • 日線方向 = 大趨勢
    • 4H 方向 = 中期動能
    • 1H 方向 = 短線節奏
    • 三時框同向 = 最強信號；矛盾 = 降低倉位
  • 關鍵支撐/壓力位標記 — 畫出今日最重要的 2-3 個價位
  • 今日 24h 預估區間（from ATR）
    預估範圍 = 當前價格 +/- 1x Daily ATR(14)
  • 布林通道位置 — Squeeze（收斂）= 大行情即將到來
▶

3. 籌碼面快掃（5 分鐘）

加密專用
  • BTC/ETH Funding Rate
    • 極正（>0.05%/8h）= 多頭過擠，注意回調
    • 極負（<-0.03%/8h）= 空頭過擠，注意軋空
  • Open Interest 變化 — OI 急升 + 價格不動 = 即將大波動
  • Fear & Greed 指數 — 極恐懼時考慮做多；極貪婪時考慮做空
  • 異常清算/大額轉帳 — 大規模清算後通常有反彈機會
▶

4. 制定今日計劃（5 分鐘）

核心

必須回答的問題：

  1. 今日交易方向 — 做多 / 做空 / 觀望（三選一，不要含糊）
  2. 最多開幾個倉位 — 建議 1-2 個，最多 3 個
  3. 每筆最大虧損額度 — 帳戶的 1-2%
  4. IF-THEN 計劃（寫下來！）：
IF 價格到達 [具體價位]
THEN 我做 [具體動作：買入/賣出/加倉]
止損設在 [具體價位]
目標設在 [具體價位]
倉位大小 = [具體數量]
沒有 IF-THEN 計劃 = 今天不交易。如果你無法寫出具體的進出場條件，代表你還沒準備好。
▶

5. 心態檢查（2 分鐘）

最重要

誠實回答自己：

問題如果是行動
今天情緒狀態如何？ 疲倦/焦慮/憤怒 不交易，或倉位減半
是否連續虧損中？ 連虧 2+ 筆 倉位降低 50%
有沒有「一定要賺回來」的想法？ 有 絕對不交易（最危險信號）
白天工作壓力大嗎？ 很大 只做觀察，不開倉
是否有足夠睡眠？ < 6 小時 降低複雜度，只做最簡單策略
報復性交易是帳戶殺手 #1。當你有「要把虧損賺回來」的念頭時，你的判斷力已經被情緒劫持。最好的交易就是不交易。

日內交易戰術

以下是適合你時段（21:30-23:30）的五種核心日內策略，以及最重要的第六種：不交易。

核心原則：少即是多 — 一週做 3-5 筆精選交易 > 每天 20 筆亂做。每種策略都有特定的適用環境。
▶

1. Opening Range Breakout (ORB)

趨勢日

定義

開盤前 15-30 分鐘（21:30-22:00）建立的高低範圍。

策略規則

  • 突破 OR 高點 + 收盤確認 → 做多
  • 突破 OR 低點 + 收盤確認 → 做空
  • 止損：OR 範圍的另一端（或中間線）
  • 目標：1x OR 範圍 為第一目標，2x 為第二目標

過濾條件（必須全部滿足）

  1. 大方向一致（SuperTrend 日線/4H 同向）
  2. 突破時成交量 > 前 5 根平均 1.5 倍
  3. 不在重大數據公布後 15 分鐘內

適用環境

趨勢明確的日子（SuperTrend 三時框同向）、大數據後方向已確認。

不適用

窄幅震盪日、VIX spike 日、重大不確定事件前。

▶

2. VWAP 策略

趨勢回調

什麼是 VWAP

VWAP = Volume Weighted Average Price（成交量加權平均價）。代表日內的「公平價格」。機構交易員用它來判斷買入/賣出是否划算。

核心規則

  • 價格持續在 VWAP 之上 = 日內偏多（多方控制）
  • 價格持續在 VWAP 之下 = 日內偏空（空方控制）
  • 回測 VWAP 入場 = 趨勢日中最安全的入場方式

策略

  1. 確認日內方向（價格 > VWAP 且保持）
  2. 等待價格回測 VWAP（不要追高）
  3. 在 VWAP 附近出現反轉 K 線時入場
  4. 止損：VWAP 另一側 + buffer
  5. 目標：前高/前低 或 VWAP 上下一個標準差
VWAP 策略的優勢：風險小（止損近）、勝率高（順勢入場）、適合上班族（不用一直盯盤，設好條件單即可）。
▶

3. 動能交易 (Momentum)

高波動

適用情境

  • 經濟數據大幅超預期
  • 財報爆發（beat + raise guidance）
  • 突發重大新聞（政策轉向、併購、黑天鵝）

入場條件

  1. 放量突破關鍵位（成交量 > 2x 平均）
  2. 等待第一次回踩（不追第一根大陽/大陰）
  3. 回踩不破前低/高 → 入場
  4. 確認後可加倉（金字塔法）

風控

  • 止損：回踩低點下方
  • 止盈：用 trailing stop（ATR x 2）跟隨
  • 絕對禁止：追高被套後不止損、無止損裸追
Momentum 交易的陷阱：看到大漲就想追。正確做法是等回踩確認。如果已經漲了 5%+ 才看到，通常已經太晚。寧可錯過也不要被套。
▶

4. 區間交易 (Range Trading)

震盪日

適用情境

  • 無明確趨勢 / 等待重大事件前
  • VIX 在 15 以下（低波動）
  • 布林通道收縮但尚未突破

策略規則

  1. 定義區間：前日高低 / 布林通道上下軌 / 明確支撐壓力
  2. 上緣做空（配合反轉信號）
  3. 下緣做多（配合反轉信號）
  4. 中間不操作！
  5. 止損：區間外 + 1x ATR

最大風險

區間突破時會被大幅止損。解法：

  • 倉位小（因為止損相對寬）
  • 重大數據/事件前關閉區間交易
  • 布林 squeeze 太久了 → 預期突破，不做區間
▶

6. 不交易的情境（最重要的策略）

必讀
市場永遠在 — 錯過不是虧損，衝動才是。

強制停止交易的條件：

情境行動恢復條件
連虧 3 筆 強制停止今日交易 明天重新開始，倉位減半
VIX spike + 方向不確定 全部觀望 等 VIX 穩定 + 方向明確
重大數據前 30 分鐘 不開新倉 數據消化 15 分鐘後
情緒失控 關掉螢幕 至少休息一天
日虧損達帳戶 3% 今日結束 明天重新評估
連虧 3 天 休息 1-2 天 完成復盤後才回來
頂尖交易者的秘密：他們大部分時間都在等待。70% 的時間觀察，20% 準備，10% 執行。一年只需要抓住幾十個好機會就夠了。

五日交易週期

每一週都有其固有的節奏。理解這個節奏可以幫助你在對的日子用力、錯的日子休息。

核心原則：復盤比交易更重要 — 不復盤的交易 = 賭博。每週的結構讓你有系統地成長。
▶

週一：定調日

觀察為主

特徵

  • 承接上週趨勢或反轉（週末消息消化）
  • 波動通常較小（機構還在評估）
  • Gap 容易回補（除非有重大消息）

任務清單

  1. 看上週五收盤後的 gap 方向和大小
  2. 判斷本週主線：延續 or 反轉
  3. 確認本週關鍵事件日曆
  4. 設定觀察清單（本週想交易的標的）

倉位建議

最大 30% 正常倉位。通常不重倉 — 等方向確認。

▶

週二/三：主攻日

最大機會

特徵

  • 本週真正趨勢在這兩天展開
  • 重要中段數據（ISM/JOLTS/ADP）
  • 機構定調本週方向

策略

  1. 方向確認後建中期倉位
  2. 允許最大倉位（100%）
  3. 趨勢明確 → ORB 或 Momentum 策略
  4. 如果週一判斷正確，週二加碼

注意事項

  • 即使是主攻日，依然要遵守風控規則
  • 最大倉位 ≠ 不設止損
  • 如果方向不明確，寧可等到週三
▶

週四：延續或預備日

管理

特徵

  • 延續週二三方向 or 開始消化
  • 初請失業金數據（20:30 台灣時間）
  • 週五前的最後完整交易日

決策框架

  1. 持有中期倉位？→ 決定週五前是否減倉
  2. 趨勢是否開始減弱？（成交量下降、動能消退）
  3. 週五有重大事件嗎？（NFP/期權到期）→ 如果有，考慮減倉

倉位建議

最大 70% 正常倉位。如果持有隔夜倉，設好防守位。

▶

週五：結算日

謹慎

特徵

  • NFP（每月第一週五）— 本月最大波動日
  • 期權到期 — Max Pain 引力強
  • 下午尾盤：機構調倉 + position squaring

策略

  • 如果無把握，週五不開新倉
  • NFP 日：只做事件驅動策略（數據後 15 分鐘入場）
  • 午後回歸 Max Pain — 不要跟這個力量作對

週末風險考量

  • 加密 24/7 運作但美股週末關閉
  • 留美股倉位 → 週一 gap 風險
  • 留加密倉位 → 週末流動性差，但有時有獨立行情
  • Funding rate 成本累積
經驗法則：如果你不願意在一整個週末不看盤的情況下持有這個倉位，那就不要留倉過週末。
▶

週末：復盤與準備

成長

週六：本週交易復盤（見 Tab 6）

  1. 統計本週所有交易的勝率和總 R
  2. 找出本週最好和最差的交易
  3. 檢查是否有違反規則的行為
  4. 情緒日誌回顧

週日：下週準備

  1. 下週重要事件日曆（數據/財報/FOMC）
  2. 各標的週線技術面分析
  3. 調整策略（如果本週某策略不適用）
  4. 設定下週觀察清單和 watchlist

風險管理

風險管理不是可選的附加功能 — 它是交易的全部。沒有風險管理的交易就是賭博。

活著才能交易明天。一次爆倉可以毀掉一年的努力。保護本金永遠是第一優先。
▶

1. 倉位大小計算

核心公式

固定風險法（推薦）

每筆交易最多虧損帳戶的 1-2%。不管多有信心，都不超過 2%。

倉位大小 = (帳戶資金 x 風險百分比) / (入場價 - 止損價)

範例：
帳戶 = $10,000
風險 = 1% = $100
入場價 = $50
止損價 = $48
止損距離 = $2
倉位大小 = $100 / $2 = 50 股

加密合約計算（含槓桿）

保證金 = (帳戶 x 風險%) / (止損% x 槓桿倍數)

範例：
帳戶 = $10,000 | 風險 = 1% = $100
止損距離 = 2% | 槓桿 = 5x
保證金 = $100 / (2% x 5) = $100 / 10% = $1,000
實際倉位 = $1,000 x 5 = $5,000
▶

2. 止損設定原則

生死線

三種止損方法

方法設定方式適用情境
技術止損 結構失效點（支撐跌破、SuperTrend 翻轉） 大多數情境，最精確
ATR 止損 入場價 +/- 1~1.5x ATR(14) 沒有明確結構時
時間止損 入場後 X 根 K 線沒有朝預期方向動 動能交易 — 預期快速獲利
絕對禁止：移動止損往虧損方向！
「再給它一點空間」= 最常見的破產藉口。止損設好了就不要動。如果覺得止損太近，那是你入場時沒算好 — 下次改善，但這次不要動。

止損常見錯誤

  • 不設止損（「先看看再說」→ 虧損從 2% 變 20%）
  • 心理止損（「等到那個價位我就手動出」→ 到了又不出）
  • 止損太近（被正常波動洗出去）
  • 止損設在整數關口（所有人都設在那裡 → stop hunt）

3. 加倉 vs 攤平

進階

加倉（正確做法 — 金字塔法）

只在獲利時加倉。每次加倉量遞減：

第一筆：50% 計劃倉位（方向初步確認）
第二筆：30% 計劃倉位（方向進一步確認）
第三筆：20% 計劃倉位（趨勢明確加速）

每次加倉都要重新計算整體止損位！

攤平（危險做法）

在虧損時加倉 = 最快破產的方式。
「跌了這麼多一定會反彈」— 這是最危險的想法。趨勢可以持續比你想像更久。

唯一例外：事先計劃好的分批建倉

事先計劃：
價格到 A → 買入 1/3
價格到 B（更低）→ 買入 1/3
價格到 C（最低）→ 買入 1/3
整體止損設在 C 下方

這不是「攤平」— 這是計劃內的分批入場。
差別在於：事先計劃 vs 臨時決定。

4. 日最大虧損限制

硬規則
規則觸發條件行動
單筆止損 虧損達帳戶 1-2% 平倉，執行止損
日最大虧損 當日累計虧損達帳戶 3-5% 強制停止交易，明天再來
連虧規則 連虧 3 筆 今日停止 + 倉位減半
連虧 3 天 連續 3 天虧損 休息 1 天，完成復盤
週最大虧損 本週虧損達帳戶 8-10% 本週結束，等下週一
月最大虧損 本月虧損達帳戶 15-20% 停止 1 週，全面復盤
這些規則不是建議 — 是硬性限制。就像開車有速限一樣，違反的後果是災難性的。把它們寫在紙上貼在螢幕旁邊。

5. 槓桿管理（加密特別重要）

高風險

建議槓桿上限

經驗等級建議最大槓桿原因
新手（< 6 個月）2-3x還在學習，要給自己犯錯空間
中等（6-24 個月）3-5x有系統但還在優化
資深（2年+且有正期望）5-10x僅限高確信度交易

槓桿 vs 爆倉距離

槓桿倍數爆倉距離（不含手續費）BTC 日波動 3% 的風險
2x50%安全
3x33%安全
5x20%中等風險
10x10%高風險（一天大波動可能爆）
20x5%極高風險（常規波動即爆）
50x2%賭博（幾乎必爆）
100x1%瞬間爆倉
真實計算：10x 槓桿 + 10% 止損 = 100% 本金虧損。很多新手不理解這一點。槓桿放大的不只是獲利，更是虧損。

6. 風報比 (Risk:Reward)

數學

最低要求

  • 最低 R:R = 1:1.5（贏的時候比輸多 50%）
  • 理想 R:R = 1:2 以上
  • 絕對不做 R:R < 1:1 的交易

為什麼 R:R 比勝率重要

期望值 = (勝率 x 平均獲利) - (敗率 x 平均虧損)

場景 A（高勝率低 R:R）:
勝率 70%, 平均贏 $100, 平均虧 $200
期望值 = (0.7 x $100) - (0.3 x $200) = $70 - $60 = +$10 ← 勉強正

場景 B（低勝率高 R:R）:
勝率 40%, 平均贏 $250, 平均虧 $100
期望值 = (0.4 x $250) - (0.6 x $100) = $100 - $60 = +$40 ← 優秀！

結論：40% 勝率 + 1:2.5 R:R > 70% 勝率 + 1:0.5 R:R

勝率 vs R:R 損益平衡表

R:R損益平衡勝率如果勝率 45%
1:150%虧損
1:1.540%獲利
1:233%獲利
1:325%獲利
1:420%獲利
重點：如果你能維持 1:2 的 R:R，你只需要贏 34% 以上就能長期獲利。把精力放在找高 R:R 的機會，而不是追求高勝率。

交易復盤框架

復盤是交易者成長的引擎。沒有復盤，你只是在重複同樣的錯誤。

核心心法：不是每筆虧損都是錯誤。正確的交易也會虧錢。真正的錯誤是：違反規則、衝動交易、移動止損、過大倉位。按計劃執行但市場走反 = 好的虧損。

5. 復盤的核心心法

最重要

區分「好的虧損」vs「壞的虧損」

好的虧損壞的虧損
按計劃執行，但市場走反 沒有計劃就入場
止損設在合理位置，被觸發 沒設止損，或移動止損往虧損方向
倉位大小合理，虧損在預期內 倉位太大，一筆虧掉太多
等待信號確認後入場 衝動入場、追高殺低
情緒穩定，接受結果 報復性交易、情緒化操作
正確的虧損不需要自責。如果你按照計劃執行，止損被觸發了，那只是機率在運作。長期來看，正確的虧損加上正確的獲利 = 持續盈利。
真正需要自責和改善的：違反自己的規則。每一次違反規則都要認真對待，找出原因，設定防護措施。
