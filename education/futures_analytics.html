永續合約籌碼分析完全教學

OI / CVD / Funding Rate / 清算圖 — 從零到實戰的完整指南

⚠️ 使用限制

OI/CVD/Funding 等衍生品指標提供概率傾向而非確定方向。OI↑+Price↑ 代表新多頭可能性高，但也可能是空頭加碼做對沖。CVD 不可單獨使用 — 必須配合 OI 和價格方向。所有框架都有失效時期，特別是在極端事件或流動性危機中。

Open Interest (OI) 研判

1. 什麼是 OI（未平倉量）

永續合約中所有未平倉合約的總量。每個合約必然有一個多方和一個空方配對。

  • OI 增加 = 新合約被創建（一個新多頭 + 一個新空頭同時進場）
  • OI 減少 = 舊合約被關閉（多頭平倉配對空頭平倉，或者一方被清算）
  • OI 不變 = 一個新進場者接手了一個離場者的部位（如：新多頭接手舊多頭的平倉單）
核心概念：OI 反映的是市場中「未結算的槓桿」有多少。OI 越高，代表場上的賭注越大，一旦方向明確，波動也越劇烈。

2. OI + 價格四象限（核心分析框架）

這是 OI 分析中最重要的框架。透過 OI 變化方向 + 價格變化方向的組合，判斷市場中「誰在做什麼」。

OI↑ + Price↑ = 新多建倉

新資金進場做多，推動價格上漲。這是最健康的上漲結構。

實戰案例：BTC 從 $25k 漲到 $30k，OI 同步增加 20%。表示有大量新多頭在建倉，趨勢有持續動力。可以順勢做多。

OI↑ + Price↓ = 新空建倉

新資金進場做空，推動價格下跌。健康的下跌結構。

實戰案例：ETH 從 $2000 跌到 $1800，OI 增加 15%。主力新空入場壓制價格，不宜抄底，應順勢做空。

OI↓ + Price↑ = 空頭被軋 (Squeeze)

空頭被迫平倉買回，推動短期上漲。但這不是新資金進場，反彈往往不持久。

實戰案例：BTC 快速從 $28k 反彈到 $30k，但 OI 下降 10%。這是 short squeeze，空頭被軋爆。反彈結束後可能回落。

OI↓ + Price↓ = 多頭投降

多頭認輸或被清算，主動/被動平倉。槓桿被清除，接近底部訊號。

實戰案例：BTC 從 $30k 急跌到 $27k，OI 暴跌 25%。多頭大規模投降/被清算。槓桿清除後，底部往往不遠。

3. OI 絕對值 vs 變化量

絕對值的意義

  • OI 絕對值高 = 市場上的槓桿總量大 = 波動加劇的潛力大
  • 高 OI 就像「炸藥」：一旦價格朝某方向突破，清算會引發級聯反應
  • 歷史 ATH（All Time High）的 OI 水位是特別危險的信號

變化量才是關鍵

  • OI 突然暴增 + 價格方向明確 = 判斷誰在建倉（配合四象限）
  • OI 突然暴降 = 大規模平倉或清算正在發生
  • OI 緩慢上升 = 市場參與度增加，醞釀大波動

4. OI 的清算含義

核心規則：OI 高 + 價格快速反向 = 大規模清算即將發生。清算是級聯的：一個被清算 → 價格惡化 → 更多被清算 → 瀑布式崩盤/暴漲。
  • 高 OI 是「燃料」，價格方向是「火柴」
  • 100x 槓桿的合約只需 1% 的反向波動就會被清算
  • 清算單是市場單（Market Order），會直接衝擊價格
  • Binance BTC 永續的清算引擎是最大的「鯨魚」

5. Coinglass OI 數據解讀

全市場 OI vs 單交易所 OI

  • 全市場 OI：反映整體市場的槓桿水位
  • Binance OI：散戶為主，OI 佔比最大（通常 40-50%）
  • Bybit OI：專業交易者偏多
  • CME OI：機構資金，反映華爾街態度

OI-weighted Funding

不同交易所的 Funding Rate 不同，而 OI 佔比最大的交易所的 Funding 對價格影響最大。Coinglass 的 OI-weighted funding 給出加權平均值，更有參考意義。

實戰技巧：如果 Binance（OI 最大）的 Funding 是 0.08%，而 Bybit 是 0.02%，代表散戶過度做多。此時 Binance 的清算風險最高。

CVD（Cumulative Volume Delta）

1. 什麼是 CVD

CVD = 累計（主動買入量 - 主動賣出量）。它衡量的是「急著交易」的人的方向。

  • Taker Buy = 買方主動吃 Ask（買方等不及，願意付更高價 = 真正急著買）
  • Taker Sell = 賣方主動打 Bid（賣方等不及，願意接受更低價 = 真正急著賣）
  • CVD = Σ(Taker Buy - Taker Sell)，累計所有差值
CVD vs 單純成交量：成交量只告訴你「交易活躍度」，CVD 告訴你「誰更急著交易」。一個急著買的人比 10 個掛單等待的人更能推動價格。

2. CVD 趨勢解讀

  • CVD 上升趨勢 = 主動買入持續大於主動賣出 = 市場上真正有人在買（不只是被動掛單）
  • CVD 下降趨勢 = 主動賣出持續大於主動買入 = 真正有人在拋售
  • CVD 走平 = 買賣平衡，市場進入觀望

3. CVD 背離（最強信號）

CVD 背離是判斷假突破和趨勢反轉最有力的工具之一。

頂背離 (Bearish Divergence)

價格創新高 but CVD 沒有創新高。代表雖然價格在漲，但主動買入力道在減弱。價格靠被動單（Maker/限價單）撐著，主力正在暗中出貨。

操作：減倉或準備做空。

底背離 (Bullish Divergence)

價格創新低 but CVD 沒有創新低。代表雖然價格在跌，但主動賣出力道在減弱。主力正在暗中吸籌。

操作：準備做多或觀察反轉訊號。

判斷假突破的利器：價格突破前高，但 CVD 沒有同步突破 = 高機率是假突破。真正的突破需要 CVD 共振確認。

4. CVD + OI 組合判讀

CVD OI Price 判讀 操作建議
CVD ↑ OI ↑ Price ↑ 最健康上漲 真買 + 新多建倉，順勢做多
CVD ↓ OI ↑ Price ↓ 最健康下跌 真賣 + 新空建倉，順勢做空
CVD ↑ OI ↓ Price ↑ Short Squeeze 空頭被迫買回，反彈不持久
CVD ↓ - Price ↑ 危險！ 有人在拋售但價格還沒反映，即將反轉
CVD ↑ - Price ↓ 底部訊號 有人在吸籌但價格還沒反映，準備反彈

5. CVD 在 Coinglass 的使用

Spot CVD vs Futures CVD

  • Spot CVD（現貨）：更反映真實需求。現貨買入 = 真金白銀購買，沒有槓桿和清算問題。
  • Futures CVD（合約）：反映投機情緒。合約的 CVD 可能被高槓桿放大。
  • 關鍵判讀：Spot CVD 上升 + Futures CVD 上升 = 最強信號（真需求 + 投機都看多）
  • 警訊：Spot CVD 下降 + Futures CVD 上升 = 泡沫（真需求減少但投機加倍看多）
Coinglass 操作提示：在 Coinglass 的 Spot/Futures CVD 頁面，可以選擇交易所和時間框架。建議同時觀察 Binance Spot CVD 和全市場 Futures CVD 的趨勢是否同步。

Funding Rate 深度解析

1. Funding 機制原理

永續合約沒有到期日（不像期貨有交割日），靠 Funding Rate 機制來讓永續價格錨定現貨價。

  • Funding > 0 = 永續價 > 現貨價 = 多方付費給空方 = 做多成本高 = 市場偏多
  • Funding < 0 = 永續價 < 現貨價 = 空方付費給多方 = 做空成本高 = 市場偏空
  • 結算頻率：每 8 小時結算一次（00:00 / 08:00 / 16:00 UTC）

2. Funding 極端值的意義

Funding 範圍 市場狀態 歷史統計 操作建議
> 0.1% 極度過熱 70% 機率 24H 內回調 5%+ 高風險做多區，考慮獲利了結
0.05% ~ 0.1% 過熱 60% 機率出現回調 收緊停損，不追高
0.01% ~ 0.05% 正常偏多 趨勢中正常 可以持有多倉
-0.01% ~ 0.01% 中性 方向由現貨決定 觀察其他指標
-0.05% ~ -0.01% 偏空 空頭佔優但非極端 可以持有空倉
< -0.05% 極度過冷 65% 機率出現 squeeze 反彈 空頭擁擠，反向做多勝率高

3. Funding 與 OI 配合

高 Funding + 高 OI = 最危險

多頭結構過度擁擠。大量高槓桿多頭在場上，每 8 小時還要付費。一旦下跌 = 踩踏式崩盤。

歷史案例：2021/4/18 BTC $64k → $51k (一天 -20%)

負 Funding + 高 OI = 空頭擁擠

大量空頭在場上且在付費做空。一旦反彈觸發清算，squeeze 動力非常強。是逆向做多的好時機。

低 Funding + 低 OI = 冷淡觸底

市場無人問津，槓桿被清除。這往往是大週期底部區域。耐心等待 OI 回升 + Funding 轉正 = 新一輪行情啟動。

高 Funding + 低 OI = 小範圍過熱

少量多頭佔據市場，Funding 高但 OI 低。清算風險有限，但仍需警惕短期回調。

4. Funding Rate 套利

當 Funding 持續為正且較高時，可以進行「資金費率套利」：

策略：永續合約做空 + 現貨做多 = Delta Neutral（市場漲跌都無風險）+ 收取 Funding 費

年化收益計算：
假設平均 Funding = 0.03% / 8H
日收益 = 0.03% x 3 = 0.09%
年化 = 0.09% x 365 = 32.85%

風險注意

  • Funding 可能突然轉負 = 從收錢變成付錢
  • 永續做空的保證金需要足夠，避免被清算
  • 現貨要放在同交易所避免出金延遲風險
  • 適合 Funding 持續高位的牛市環境

清算圖與 Long/Short Ratio

1. 清算熱力圖 (Liquidation Heatmap)

清算熱力圖顯示在不同價位上，有多少槓桿部位會被清算。顏色越深 = 清算量越大 = 價格磁鐵越強。

清算密集區 = 價格磁鐵。價格傾向被吸引到清算密集區，因為觸發清算會產生更多動能（正反饋），這等於「免費流動性」，主力最喜歡在這裡操作。

主力的策略

  1. 觀察清算熱力圖，找到清算密集的價格區域
  2. 推動價格到該區域，觸發大量清算
  3. 清算產生大量市場單 = 極好的流動性
  4. 在這個流動性中反向建倉（別人被清算時你進場）

2. Long/Short Ratio

Long/Short Ratio 顯示散戶帳戶中做多 vs 做空的比例。注意：這是「帳戶數量」比，不是「金額」比。

使用注意：L/S Ratio 是帳戶數量比，不是資金量比。100 個小散戶做多 vs 1 個鯨魚做空，L/S 顯示多頭佔優但實際空方力量更大。因此這是「反向指標」——散戶大量做多時反而要小心。

3. 清算級聯 (Cascade)

加密貨幣市場的波動比傳統市場大的核心原因之一就是清算級聯機制。

為什麼加密市場波動比傳統市場大？

  • 100x 槓桿：1% 波動就清算，傳統市場最高 5-10x
  • 24/7 交易：沒有熔斷機制，半夜也能崩盤
  • 無漲跌停：一天跌 30-50% 完全可能
  • 級聯清算：正反饋循環加劇波動
  • 交叉交易所：A 所清算 → 套利者搬磚 → B 所也跌 → B 所清算

4. 實戰應用：用清算圖找交易機會

情境 A：下方有大量多頭清算

如果當前價 $30k，而 $28k-$29k 有大量多頭清算密集區：

  • 價格可能被「吸下去」觸發清算
  • 清算完成後（OI 暴降），反而是底部
  • 策略：在清算區下方設限價買單，等待 V 反

情境 B：上方有大量空頭清算

如果當前價 $30k，而 $31k-$32k 有大量空頭清算密集區：

  • 價格可能被「軋上去」觸發 squeeze
  • Squeeze 完成後（空頭被清除），上漲動力消失
  • 策略：順勢做多搭車，但在清算區頂部附近獲利了結
Coinglass 清算圖使用技巧：
  1. 選擇 BTC/USDT 永續合約
  2. 觀察清算密度最高的價格區域（顏色最深處）
  3. 距離當前價越近的清算密集區 = 越容易被觸發 = 短期價格方向
  4. 距離當前價較遠的清算密集區 = 中期目標位

完整籌碼分析 SOP

1. 五步驟判讀流程

1

看 OI 變化方向

OI 增加 = 新倉進場（新合約被創建）
OI 減少 = 舊倉離場（合約被關閉）
判斷是「新錢進場」還是「舊錢離場」

2

配合價格方向 - 四象限定位

OI↑+Price↑ = 新多 | OI↑+Price↓ = 新空 | OI↓+Price↑ = Squeeze | OI↓+Price↓ = 投降
定位當前市場在哪個象限，判斷趨勢性質

3

看 CVD 確認真買/真賣 or 被動

CVD 同向 = 確認（真正有人主動交易）
CVD 背離 = 假突破/即將反轉
CVD 是判斷「力道真假」的關鍵

4

看 Funding 判斷市場溫度

Funding 極端正 = 過熱 = 回調風險
Funding 極端負 = 過冷 = 反彈動力
Funding 中性 = 健康趨勢中

5

看清算圖找短期價格磁鐵

上方清算密集 = 可能被軋上去
下方清算密集 = 可能被吸下去
找到最近的清算密集區 = 短期目標位

3. 不同時間框架的 OI/CVD

短線 (1-4H)

關注：Funding Rate + OI 快速變化

用途：Scalping 入場點 / 短期方向

技巧：Funding 結算前後 OI 變化常產生短線機會（結算前平倉 → 結算後重建）

中線 (日線)

關注：OI 趨勢 + CVD 背離

用途：Swing Trade 方向判斷

技巧：日線 CVD 背離是最可靠的反轉信號之一，配合 OI 四象限做方向判斷

長線 (週線)

關注：總 OI 水位 + Funding 均值

用途：大週期頂底判斷

技巧：OI ATH + Funding 長期高位 = 週期頂部。OI 極低 + Funding 長期負值 = 週期底部

多時間框架整合

原則：大時框定方向，小時框找入場

週線判斷牛/熊 → 日線找趨勢 → 4H 找進場點

衝突時：以大時框為準，不逆勢操作

4. 完整案例分析：BTC 假設場景

場景設定：BTC 從 $28,000 連漲到 $32,000，你需要決定是否繼續做多。
1

Step 1 - OI 分析

過去 3 天 OI 從 $12B 增加到 $14.5B（+20%）。
判讀：大量新倉進場。

2

Step 2 - 四象限定位

OI ↑ + Price ↑ = 新多建倉象限。
判讀：趨勢健康，有新資金做多推動。

3

Step 3 - CVD 確認

CVD 在前兩天同步上升，但今天價格創新高 $32k 時 CVD 卻沒有跟上（頂背離出現）。
警訊！主動買入力道減弱，價格靠被動單撐。

4

Step 4 - Funding 判斷

Funding Rate = 0.08%（極端偏高）。
警訊！做多成本高，市場過熱。多頭需要每 8 小時付 0.08%。

5

Step 5 - 清算圖

$30k-$30.5k 有大量多頭清算密集區。上方 $33k 有一些空頭清算但量不大。
判讀：下方吸引力 > 上方吸引力。

綜合結論：
雖然 Step 1-2 顯示上漲健康，但 Step 3-5 亮起紅燈：CVD 頂背離 + Funding 過熱 + 下方清算密集。

操作決策：不追多！收緊停損到 $30.5k（清算密集區上方）。如果價格跌破 $31k 且 OI 開始下降 = 確認反轉，減倉或反手做空。
