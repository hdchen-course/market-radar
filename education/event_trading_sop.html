核心規則：Tier 1 事件前不持有大倉位過夜。違反這條 = 賭博而非交易。
你的白銀爆倉就是持倉過 CPI 的結果。這不是技術問題，是紀律問題。

事件前 SOP — 分階段減倉

T-24h（事件前24小時）：確認事件存在
  • 查經濟日曆（investing.com/economic-calendar）確認明天有什麼事件
  • 判定 Tier 1/2/3
  • 如果是 Tier 1 → 開始計劃減倉
T-4h（事件前4小時）：執行減倉
  • Tier 1 事件：倉位減至 25% 或更少
  • Tier 2 事件：倉位減至 50%
  • 確認止損單已設（在 ST level 或更寬的位置）
  • 如果當前有浮盈 → 先鎖利潤再說
T-30min（事件前30分鐘）：最終確認
  • 確認市場共識是什麼（Bloomberg/Investing.com forecast）
  • 如果倉位仍大 → 市價平一半，不等好價格
  • 不要在這時候開新倉！
T-0（事件公布瞬間）：不動
  • 不操作。看數據。看市場反應。
  • 前 5 分鐘的波動 90% 是噪音
  • 等 15-30 分鐘方向明確再行動
決策流程：
┌─ 我有倉位嗎？
│   ├─ No → 不做任何事，等事件後再進場
│   └─ Yes → 24小時內有 Tier 1 事件嗎？
│         ├─ Yes → 減至 25% 或空手。設止損。
│         └─ No → Tier 2？
│               ├─ Yes → 減至 50%。設止損。
│               └─ No → 正常操作
└─ 我有浮盈嗎？
     ├─ Yes → 先平一半鎖利潤。剩下設trailing stop。
     └─ No (浮虧) → 考慮全平。不要帶虧損倉位過事件。
睡覺前的鐵律：
如果明天有 Tier 1 事件且你的時區會在睡覺時公布（FOMC 凌晨2:00）→ 睡前必須空手或設好止損。你醒來時可能已經爆倉了。（白銀 $65→$61.53 就是睡著時發生的）

事件中反應 — 前5分鐘是噪音

數據公布後的市場節奏

時間發生什麼你該做什麼
0-5 分鐘演算法交易主導。巨幅波動。假突破。插針。不動。不追。不搶。
5-15 分鐘第一波方向初步形成。但可能反轉。觀察方向。不進場。
15-30 分鐘真正的方向開始確認。成交量歸位。如果 ST 1h 翻轉確認方向 → 可以進場小倉
30-60 分鐘「第二波」movement 通常更可靠加倉或確認方向正確

如何解讀「符合預期但仍暴跌」

情境解讀操作
數據好 + 資產漲正常反應。跟隨。順勢做多
數據好 + 資產跌「Buy the rumor, sell the news」已 price in做空或觀望
數據差 + 資產跌正常反應。跟隨。順勢做空
數據差 + 資產漲超賣反彈 or 市場認為Fed會因此轉鴿觀望→確認後跟多
符合預期 + 暴跌預期本身就很糟（如CPI 4.2%「符合」但太高）跟跌方向。「符合壞預期」≠ 好事
符合預期 + 暴漲市場之前過度恐慌，relief rally跟漲但設緊止損（可能是死貓跳）
絕對不要做的事：
  • 不要在數據公布的前5秒下單（滑點巨大）
  • 不要追第一根大K線（90%會回調）
  • 不要因為「判斷正確」就重倉（方向對但入場太早=被洗出去）
  • 不要移動止損（事件波動大，止損被掃是正常的，但沒止損是災難）

事件後操作 — 等「第二波」

「第二波」比「第一波」更可靠

事件後的第一波反應往往過度（演算法驅動）。真正有方向性的是 30 分鐘後到收盤前的「第二波」。

第一波 vs 第二波第一波（0-15min）第二波（30min-收盤）
驅動者演算法、止損觸發、恐慌機構決策、真實資金流
可靠度低（常被反轉）高（通常持續到收盤）
入場風險高滑點、假突破多滑點正常、方向較確定
適合操作不適合適合小倉位跟隨

事件後的進場 SOP

等 15-30 分鐘 — 讓噪音消退
確認 1h SuperTrend 方向 — 如果跟數據暗示的方向一致 → 信號可靠
小倉位入場（最多 20%）— 事件後波動仍大，不重倉
止損設在事件後的極值 — 例如 CPI 後最低點（多單）或最高點（空單）
確認 4h ST 後加倉 — 如果 4h 也翻轉同方向 → 倉位加到 30-40%

「Sell the News」模式識別

當市場在事件前已經大幅反應（price in），事件公布後反而反方向走。

識別信號案例
事件前連續上漲 + 事件後公布利多NFP超強→本應利多美元→但美元反跌（已price in）
事件前VIX持續升高 + 結果符合預期CPI 4.2%符合→恐慌已消化→反彈（relief rally）
大量 OTM Call/Put 在事件前被買入期權市場已經定位→事件後gamma收斂→方向反轉
