交易心理學 — 你的大腦才是最大的敵人

技術分析只佔 20%，心理紀律佔 80%。90% 散戶虧損不是因為不懂技術，而是因為無法控制情緒。（此為業界常引數據，來源包括 eToro 2020 年報告及各券商風險揭露，實際比例因市場/時期而異）

認知偏誤：大腦如何欺騙你

損失厭惡Loss Aversion
研究顯示：虧損帶來的痛苦是等額獲利帶來快樂的 2.5 倍。這導致兩個致命行為：(1) 不願意停損，因為「確認虧損」太痛苦；(2) 過早獲利了結，因為害怕利潤消失。
場景：你做多 ETH 賺了 $200，開始害怕回吐而平倉。但你做空 BTC 虧了 $500，卻死扛著說「會回來的」。贏小虧大，長期必輸。
確認偏誤Confirmation Bias
一旦你有了方向偏見（看多或看空），你會只搜尋支持你觀點的資訊，忽略所有反向信號。多頭持倉時只看利多新聞，忽略即將到期的巨額選擇權壓力。
場景：你做多黃金，然後只關注地緣風險新聞，完全忽略了美元指數正在暴力上漲、實質利率飆升的事實。結果被反向收割。
錨定效應Anchoring
你的大腦會被入場價格綁架。無論市場結構如何變化，你都只在乎「回到我的成本價」。這個「錨」讓你無法理性評估當前市場狀態。
場景：你在 $72K 買了 BTC，跌到 $65K。你不斷告訴自己「等回到 $72K 我就出」。但市場結構已經破壞，正確做法是現在就止損，而非等待一個可能永遠不會來的價格。
近因偏誤Recency Bias
最近發生的事件被賦予不成比例的權重。連贏 3 筆後你覺得自己無所不能，連虧 3 筆後你覺得市場在針對你。兩者都是幻覺。
場景：你連贏 5 筆做多交易，開始覺得「空頭都是傻子」。然後趨勢反轉，你不相信市場會跌，因為你最近的經驗全是漲。結果一筆虧掉前五筆的利潤。
沉沒成本Sunk Cost Fallacy
「已經虧了這麼多，現在止損就白虧了」— 這是典型的沉沒成本謬誤。已經虧的錢永遠不會因為你繼續持有而回來，但你可能虧更多。
場景：你做多某山寨幣，已經虧了 40%。你心想「都虧這麼多了，再等等吧」。結果從虧 40% 變成虧 80%。如果當初止損在 -10%，只是一個小傷口。
過度自信Overconfidence Bias
連勝會讓大腦分泌多巴胺，產生「我已經看透市場」的幻覺。這時你會加大倉位、放寬止損、忽略風險管理。爆倉通常發生在連勝之後，而非連虧之後。
場景：連贏 5 筆後，你決定把平常 1% 風險的倉位加到 5%。「反正最近手感很好」。結果這筆虧了，一次虧掉之前五筆利潤的兩倍。帳戶回到原點以下。
從眾效應Herd Mentality
看到 KOL 或群組裡所有人都在做多，你會覺得「這麼多人不可能都錯」。但散戶共識通常是完美的反指標 — 當所有人都看多時，已經沒有新的買盤了。
場景：你的交易群組裡所有人都在曬做多獲利，KOL 說「BTC 下一站 $100K」。你跟著入場。結果你就是最後一棒，聰明錢正在把貨出給你。

情緒循環：散戶的完美反指標

關鍵認知：市場的頂部不是由「壞消息」製造的 — 而是由「所有好消息都已被定價」的共識製造的。當你覺得最興奮時，通常就是最危險的時候。聰明錢在你恐懼時買入，在你貪婪時賣出。

六大虧損模式：你中了幾個？

1. 報復性交易Revenge Trading
心理機制
虧損觸發「戰鬥或逃跑」反應，大腦進入原始模式。杏仁核接管決策，前額葉皮質（理性思考）被抑制。你不是在交易，你是在和市場「打架」。
典型場景
你做空 BTC 被止損虧了 $500。15 秒後你就開了一個 2 倍倉位的做多單，沒有任何分析，只有一個念頭：「馬上賺回來」。
後果
因為是情緒決策而非理性分析，這筆交易通常也會虧。結果一天之內從虧 $500 變成虧 $1500。更糟的是，你可能繼續報復，直到帳戶嚴重受損。
解決方案：
1. 虧損後強制離開螢幕至少 15 分鐘（設手機鬧鐘）
2. 設定日最大虧損 2%，觸發後強制停手整天
3. 連虧 2 筆 = 休息 30 分鐘再考慮是否繼續
4. 在交易日記寫下：「報復性交易從未讓我賺錢」
自我檢查：「這筆交易是我計劃中的設定，還是因為剛才虧錢所以想趕快賺回來？」— 如果答案是後者，立即關閉下單介面。
2. FOMOFear of Missing Out
心理機制
看到別人賺錢而自己沒有參與，觸發社交比較焦慮。大腦把「沒賺到的利潤」等同於「實際的損失」，產生緊迫感驅使你追入。
典型場景
某 token 連漲 3 天 80%，你的 Twitter 和群組充滿獲利截圖。你心想：「再不上車就來不及了」。於是在已經漲了 80% 之後，你追入了。
後果
你成為最後一棒接盤俠。聰明錢需要流動性來出貨，而 FOMO 散戶提供了完美的出貨對象。價格在你入場後反轉，你被套在最高點。
解決方案：
1. 核心認知：「沒上車的機會不是損失，追高才是真正的損失」
2. 錯過了就等下一個回調入場機會，不追高
3. 問自己：「如果這個幣現在才 ICO，以這個價格我會買嗎？」
4. 退出那些只曬獲利不曬虧損的群組
自我檢查：「我是因為自己的分析看到機會，還是因為別人賺了錢所以我也想要？」— 如果是後者，這不是交易信號，這是社交焦慮。
3. 不停損Holding Losers
心理機制
損失厭惡讓你無法接受「確認虧損」。只要不平倉，虧損就只是「帳面虧損」— 大腦用這個方式欺騙自己。同時，「希望」這個最危險的情緒會不斷告訴你「再等等就會回來了」。
典型場景
你設了 $71.5K 的止損，BTC 跌到 $71.3K。你心想「差一點點就止損了，再觀察一下」。於是你把止損移到 $72K... 然後 $73K... 然後乾脆刪掉止損。BTC 跌到 $68K，你還在扛。
後果
小虧變大虧，大虧變爆倉。一筆原本只虧 1% 的交易，最後虧了 15%。更嚴重的是，大額浮虧佔據你的心智資源，影響所有其他決策。
解決方案：
1. 止損必須在下單同時設定，且設定後不得移動（只能往獲利方向移）
2. 用 OCO 或條件單，讓止損是自動執行的
3. 重新定義止損：它不是「虧損」，它是「保險費」
4. 接受：止損是交易的一部分，就像呼吸是生命的一部分
自我檢查：「如果我現在沒有持倉，以當前的價格和市場結構，我會在這裡入場嗎？」— 如果答案是否定的，那你就不應該繼續持有。
4. 過度交易Overtrading
心理機制
交易會觸發多巴胺系統，和賭博機制相同。大腦渴望「行動」的刺激，靜坐不動讓你焦慮。你告訴自己「應該在做些什麼」，把「交易量」和「努力」劃上等號。
典型場景
今天沒有明確的設定出現，但你已經盯盤兩小時了，覺得「不能白坐」。於是你找了一個勉強算有的信號就入場了。一天做了 12 筆，其中 9 筆不在計劃中。
後果
手續費和滑點慢慢侵蝕帳戶。勝率下降因為你在交易低品質的設定。精力分散導致真正的好機會出現時反而沒注意到或已經耗盡心力。
解決方案：
1. 設定每日最大交易筆數：3-5 筆
2. 「不交易也是一種策略」— 最好的交易者大部分時間都在等待
3. 離開盤面：只在固定時段看盤（如 21:30-23:30）
4. 記錄哪些交易是「計劃中」vs「衝動的」，計算兩者勝率差
自我檢查：「這筆交易是因為出現了 A+ 設定，還是因為我無聊/焦慮/想要『做些什麼』？」— 無聊不是入場理由。
5. 倉位過大Overleveraging
心理機制
「這次特別確定」的感覺觸發過度自信。或者之前虧了想「一筆翻本」。大腦把「高倉位」和「快速致富」連結，忽略了同時也是「快速破產」。
典型場景
你「非常確定」BTC 會漲（因為你看了三個指標都顯示做多）。於是你用了 10 倍槓桿全倉入場。BTC 波動 2% 反向，你直接爆倉。
後果
一筆交易虧損 30-50%，心態徹底崩潰。即使後續策略正確，你已經沒有足夠資金來執行。更危險的是，大虧後容易觸發報復性交易，形成死亡螺旋。
解決方案：
1. 鐵律：每筆風險永遠不超過帳戶的 1-2%
2. 「確定性」是幻覺 — 任何交易都可能虧損
3. 計算倉位大小公式：倉位 = (帳戶 × 風險%) / (入場價 - 止損價)
4. 用小帳戶練習紀律，等穩定獲利再加大本金
自我檢查：「如果這筆交易虧了最大止損，我能不能面不改色地接受？」— 如果會心痛、會影響情緒，說明倉位太大。
6. 過早獲利了結Premature Profit Taking
心理機制
一有獲利，損失厭惡開始作用 — 你害怕利潤「被拿走」。大腦把未實現利潤視為「已有的東西」，任何回調都像是「失去」。確定的小利潤比不確定的大利潤更讓大腦舒服。
典型場景
你做多 XAG，目標 $78，剛到 $76.5 就開始害怕回調。「先落袋為安吧」。結果 XAG 繼續漲到 $79。你的帳戶成績：賺 1% 就跑，虧 5% 才停損。
後果
贏小虧大 = 長期必輸。即使勝率 60%，如果平均獲利只有平均虧損的 1/3，你的期望值還是負的。你永遠在「截斷獲利、讓虧損奔跑」。
解決方案：
1. 使用 trailing stop（移動止盈）讓利潤自己跑
2. 部分平倉法：到第一目標平 1/3，到第二目標平 1/3，最後 1/3 讓它跑
3. 風報比紀律：不入場 R:R < 2:1 的交易
4. 提醒自己：「趨勢的力量永遠比你想的大」
自我檢查：「我想平倉是因為技術面信號告訴我趨勢結束了，還是因為我害怕利潤消失？」— 恐懼不是平倉的理由，反轉信號才是。

交易規則清單（寫下來，貼在螢幕旁）

核心交易規則 — 不可違反
  • 1每筆風險 ≤ 帳戶 1-2%，永遠不例外
  • 2止損必須在下單前設定，設定後不得往虧損方向移動
  • 3連虧 3 筆 = 今日停手，明天再來
  • 4日最大虧損 3% = 強制休息至少 24 小時
  • 5沒有計劃的交易 = 不做。「感覺」不是入場理由
  • 6重大數據公布前 30 分鐘 = 不開新倉
  • 7情緒不穩（剛虧損/生氣/興奮）= 不交易
  • 8每日最多交易 5 筆，超過 = 過度交易
  • 9只交易 A+ 設定（符合所有入場條件），B 級設定觀望
  • 10入場前必須能清楚說出：入場理由、止損位、目標位
重要：規則的價值在於你「想違反時依然遵守」。任何人都能在心情好時守規則。真正的紀律是在痛苦中、在虧損後、在 FOMO 中依然遵守。

IF-THEN 預設決策系統

消除即時決策壓力 — 事先規劃好所有情境

當情緒湧上來時，你沒有時間和能力做出好決策。解決方案：提前設定好所有 IF-THEN 規則，讓決策變成自動化的。

IF（觸發條件）THEN（預設行動）
IF 價格到達入場點 + K 線確認 THEN 按計劃入場，止損 Y，目標 Z
IF 止損被觸發 THEN 接受損失，不報復，記錄在日記中
IF 連虧 3 筆 THEN 關掉交易軟體，明天再來
IF 感覺「一定要做這筆」 THEN 這正是不該做的信號 — 等待冷靜後重新評估
IF 群組/KOL 說「All in」 THEN 忽略，只執行自己的計劃
IF 已獲利但想加倉 THEN 只在回調到支撐位 + 新信號確認時才加倉
IF 想把止損移遠一點 THEN 絕對不行。如果止損位不對，是當初分析有誤，平倉重來
IF 覺得「這次一定賺」想加大倉位 THEN 維持標準倉位。確定性是幻覺
IF 沒有設定出現但很想交易 THEN 去運動/讀書/散步。不交易也是策略
IF 剛大賺一筆心情很嗨 THEN 停手休息。興奮是過度自信的前兆

3. 寫下今日交易計劃（最多 3 個設定）
4. 設定 IF-THEN 規則
5. 2 分鐘冥想：「今天我只做計劃中的交易」
交易時段 (21:30-23:30)

2. 每筆下單前問：這是計劃中的嗎？

4. 不在中間盯盤（設好單就離開）

3. 計算當日紀律分數
4. 找出一個改進點

關鍵理念：不要靠「忍住不看盤」的意志力。直接設定好機械化的時間表。你不需要抵抗誘惑 — 你需要設計一個不會遇到誘惑的環境。

接受不確定性

每筆交易都可能虧損 — 即使分析完美
市場本質上是不確定的。即使所有技術指標都指向同一方向，下一刻發生什麼你無法控制。黑天鵝、央行突然干預、鯨魚清算 — 這些超出你的分析範圍。
正確的思考方式：「我做了正確的決策過程」而非「我要每筆都對」。一個有 60% 勝率的策略意味著每 10 筆中有 4 筆會虧 — 這是正常的，不是你的錯。
類比：賭場莊家的思維
賭場莊家不在乎單手勝負。他知道他有邊際優勢（如 2%），他只需要讓遊戲重複足夠多次，利潤就是確定的。

你也是一樣：
- 你的策略 = 你的「邊際優勢」
- 每筆交易 = 一手牌
- 紀律執行 = 讓遊戲不斷重複
- 長期(100+筆) = 邊際優勢必然顯現

你輸不起的不是一筆交易，而是你的紀律。

從結果思維 → 過程思維

結果思維（危險）
「我賺了 = 我做對了」
「我虧了 = 我做錯了」

問題：這讓你追求結果而非過程，導致你用結果來驗證方法。衝動追高賺了？你覺得追高是對的。嚴守紀律但虧了？你開始懷疑紀律。
過程思維（正確）
「我遵守了紀律 = 好交易」
「不管盈虧，過程正確 = 成功」

好交易可以虧錢（按規則做但市場走反）
壞交易可以賺錢（衝動追高但剛好對了）
→ 後者更危險：強化壞習慣
最危險的情況：違反紀律但賺了錢。這會讓你的大腦形成「違規 = 有獎勵」的連結，下次更容易違規。長期來看，這比「遵守紀律但虧損」傷害大得多。

情緒觸發的生理應對

當情緒升起時的即時工具包
深呼吸 4-7-8 法

原理：啟動副交感神經系統，降低心率和皮質醇。3 個循環後，杏仁核活動顯著降低。

正念 (Mindfulness) 在交易中的應用

觀察情緒而不被情緒帶走
正念的核心不是「不生氣」或「不焦慮」— 而是在情緒出現時，你能觀察它而非成為它。
被情緒控制的反應
「我好焦慮，趕快做些什麼！」
→ 衝動下單 → 虧損 → 更焦慮
正念覺察的回應
「我注意到自己正在焦慮」
→ 觀察 → 標記 → 選擇不行動
實戰應用：
盤前 2 分鐘冥想：閉眼深呼吸，設定今日意圖：「今天我只做計劃中的交易。任何衝動升起時，我會觀察它、標記它、然後讓它過去。」
衝動升起時的 S.T.O.P. 技巧：
S = Stop（停止，手離開鍵盤）
T = Take a breath（深呼吸一次）
O = Observe（觀察：我現在感覺什麼？這是什麼偏誤？）
P = Proceed（選擇：我要按計劃行動，或不行動）
情緒標記練習：當衝動升起時，大聲說出或寫下：「這是 FOMO」/「這是報復心態」/「這是過度自信」。命名它就削弱了它的力量。

機率思維

每筆交易 = 擲一枚有邊際優勢的硬幣
你不能控制這一筆的結果。你能控制的只有：策略選擇、倉位大小、止損設定、交易頻率。把這些做對，然後讓大數法則為你工作。
核心洞察：你不需要 80% 勝率。45% 勝率 + 2:1 風報比 = 穩定獲利。關鍵不是「對的次數」而是「對的時候賺多少 vs 錯的時候虧多少」。

心態轉換：當你把止損從「我虧了錢」重新定義為「我付了合理的成本來保護帳戶」，你會發現止損變得容易多了。那些不願意付「保險費」的人，最終會付出更大的「重建費」。

耐心 = 最高形式的紀律

「什麼都不做的能力」是頂尖交易者的超能力
等待 A+ 設定
不是每天都需要交易。頂尖狙擊手可能等待數小時只為一個完美射擊機會。你也應該如此 — 寧可錯過十個 B 級機會，也不要浪費子彈在低品質設定上。
等待入場點
看到趨勢方向不等於立即入場。等回調、等確認、等你的入場條件全部滿足。在信號出現前入場 = 賭博，不是交易。
等待獲利達標
設定目標後就信任你的分析。趨勢的力量通常比你想的更大。用 trailing stop 保護利潤而非過早平倉。讓利潤奔跑。
等待市場來到你面前
不追價、不 FOMO。如果你錯過了入場，就等下一次。市場永遠會提供新機會 — 但你的本金虧了就沒了。

一致性 (Consistency)

不追求暴利，追求穩定小贏
每天 +0.5% × 20 交易日 = 月 +10.5%（複利）
但前提是：控制 drawdown，永不爆倉
最大的敵人不是虧損，是「不一致」。一天嚴守紀律賺 1%，下一天衝動交易虧 3%。結果一個月下來不但沒賺還虧了。

一致性意味著：
  • ✓每天用同樣的策略、同樣的倉位規則
  • ✓不因為連勝就加大，不因為連虧就報復
  • ✓不因為「今天感覺好」就放鬆紀律
  • ✓把交易當工作而非刺激 — 無聊才是正確的狀態

成長心態 vs 固定心態

固定心態（有害）
「我是/不是一個好交易者」

每筆虧損 = 否定自我價值
每筆獲利 = 確認自我價值

結果：情緒隨盈虧劇烈波動，無法客觀學習，因為承認錯誤 = 承認自己「不行」
成長心態（有益）
「我正在成為更好的交易者」

每筆虧損 = 學習機會
每筆獲利 = 過程正確的驗證

結果：能客觀分析錯誤，持續改進，不被盈虧綁架自我認同
衡量進步的方式：不只看 P&L。追蹤你的紀律遵守率（本週 85% → 下週 90%）、衝動交易次數（本週 4 次 → 下週 2 次）、情緒恢復速度（止損後多快回到平靜）。這些才是真正的進步指標。

情境模擬訓練 — 測試你的即時反應

每個場景只有一個正確答案。選擇後會顯示每個選項的後果分析。訓練你在壓力下做出正確決策的肌肉記憶。

1止損邊緣的掙扎
你做空 BTC 在 $70,000，設了 $71,500 的止損。現在 BTC 漲到 $71,200，離你的止損只剩 $300。你看著價格一跳一跳地接近止損位，心跳加速，手心出汗。你想...
  • A. 把止損移到 $72,500，給它多一點空間
  • B. 什麼都不做，讓止損自動觸發就好
  • C. 趁還沒觸發，先手動平倉減少虧損
  • D. 在 $71,200 加倉做空，攤平成本
正確答案：B
你的止損是你在入場前經過分析設定的，代表你認為超過這個價位你的做空論點就失效了。在情緒壓力下修改止損 = 讓情緒推翻理性。讓系統自動執行。
A 的後果：移動止損 = 損失厭惡的典型表現。BTC 可能繼續漲到 $73K，你的小虧變大虧。
C 的後果：提前手動平倉表面上「減少虧損」，但長期會破壞你的系統。有時價格觸及止損後會反轉。
D 的後果：在錯誤方向加倉 = 沉沒成本謬誤 + 攤平成本幻覺。極度危險。
心理學原理：損失厭惡 + 錨定效應。你被入場價綁架，無法接受「確認虧損」。
2連虧後的誘惑
你今天已經虧了 2 筆，共虧 $800。你的日最大虧損限制是 $1000。現在你看到 ETH 出現一個不太確定的做多信號 — K 線有反轉跡象但量能不足，SuperTrend 還是空頭。你的內心在說：「也許這筆能把今天虧的賺回來...」
  • A. 入場做多，止損設嚴一點，控制在 $200 以內
  • B. 入場做多但用一半倉位，降低風險
  • C. 不交易。信號不確定 + 已經連虧 = 今日停手
  • D. 等到信號更明確再決定是否入場
正確答案：C
兩個紅旗同時出現：(1) 信號不確定（不是 A+ 設定）(2) 已經連虧 2 筆（情緒可能已受影響）。「把今天虧的賺回來」這個想法本身就是報復性交易的前兆。今日停手。
A 的後果：在不確定的設定上入場 = 降低勝率。加上連虧後的心態，你可能會提前平倉或亂移止損。
B 的後果：減少倉位不能解決根本問題 — 你的入場動機是「想賺回來」而非「信號明確」。
D 的後果：比 A/B 好，但今天你已經不在最佳狀態了。即使信號變明確，你的判斷也可能受情緒影響。
心理學原理：報復性交易 + 沉沒成本。「已經虧了就想賺回來」是最常見的連續虧損催化劑。
3FOMO 的力量
你的朋友群組裡所有人都在曬 HYPE 的獲利截圖，有人賺了 50%，有人賺了 200%。你沒有持倉。HYPE 已經連漲 3 天共 120%，現在還在創新高。群組的氣氛越來越瘋狂：「上車！別猶豫了！」
  • A. 趕快買入！已經漲了這麼多表示趨勢很強
  • B. 買入但設嚴格止損，控制風險
  • C. 等回調 20-30% 再考慮入場
  • D. 不參與。這不在我的交易計劃中，沒上車不是損失
正確答案：D
核心認知：「沒上車的機會不是損失，追高才是真正的損失」。連漲 3 天 120% 後，所有想買的人都已經買了（你的群組就是證據）。當你想追入時，你可能就是最後一批接盤俠。
A 的後果：在連漲 120% 後追入，你是在為聰明錢提供出貨流動性。極高機率被套。
B 的後果：即使有止損，在這種暴漲品種上止損很可能被跳空跌破。而且入場動機是 FOMO 不是分析。
C 的後果：比 A/B 好，但等回調可能是合理的技術策略。然而此時你的動機仍然是 FOMO，回調時你可能因為恐懼反而不敢買。
心理學原理：從眾效應 + FOMO + 社交比較焦慮。當「所有人都在賺」時，你最需要保持冷靜。
4利潤回吐的恐懼
你做多 XAG 在 $75，目標 $78（基於技術分析的前高阻力位）。現在價格到 $76.5，你已經浮盈。但剛才出現一根小陰線開始回落，你的心開始緊張：「萬一掉回去怎麼辦？先跑再說？」你的 trailing stop 設在 $75.8。
  • A. 維持原計劃，信任你的分析。Trailing stop 會保護你
  • B. 先平一半倉位落袋為安，另一半讓它跑
  • C. 全部平倉，先確保利潤到手
  • D. 把 trailing stop 移到 $76.3，鎖定更多利潤
正確答案：A
一根小陰線不是趨勢反轉信號。你的 trailing stop 已經在保護你（$75.8 已經鎖定盈虧平衡以上）。讓你的分析和系統工作。目標是 $78，讓利潤奔跑。
B 的後果：如果這是你預設的部分平倉策略就沒問題，但如果是因為恐懼而臨時起意，就是過早獲利了結。
C 的後果：典型的過早獲利了結。你會反覆經歷「跑太早」的遺憾，同時養成「賺小虧大」的習慣。
D 的後果：把 trailing stop 收太緊，正常的盤中波動可能會掃到你的止損，然後價格繼續往你原本的方向走。
心理學原理：損失厭惡。大腦把浮盈視為「已有的東西」，任何回調都像「失去」。解法：信任系統。
5連勝的陷阱
你連贏 5 筆，帳戶創新高。你感覺自己「看透了市場」，每個信號都清晰無比。你平常每筆風險 1% ($200)。現在你看到一個設定，感覺「超級確定」，想用 3 倍平常的倉位 ($600 風險) 入場。
  • A. 用 3 倍倉位入場，趁手感好多賺一點
  • B. 維持標準 1% 風險。「確定性」是幻覺，紀律不因連勝改變
  • C. 用 2 倍倉位作為折衷
  • D. 今天不交易了，連勝後最好休息冷靜一下
正確答案：B
「看透市場」的感覺 = 過度自信偏誤的典型症狀。統計上，連贏後下一筆的勝率並不會提高。但如果你加大倉位，一旦虧損就會抹平之前多筆利潤，甚至打破你的心態，觸發報復性交易。
A 的後果：如果虧了（完全可能），你一筆就虧掉前 3 筆的利潤。帳戶從新高急跌，心態很可能崩。
C 的後果：任何偏離標準倉位的行為都是紀律的破口。一旦你允許「稍微加大」，下次可能是「再大一點」。
D 的後果：休息不是壞選擇，但如果設定確實是 A+ 且你感覺平靜（不是亢奮），按標準倉位交易是正確的。
心理學原理：過度自信 + 近因偏誤。連勝分泌多巴胺，讓你覺得自己「無敵」。爆倉最常發生在連勝之後。
6恐慌中的冷靜
BTC 突然暴跌 8%，Fear & Greed Index = 11（極度恐懼）。你的分析框架顯示這可能是超賣底部區域（RSI < 25，偏離布林帶下軌），但 SuperTrend 全時間框架都是空頭信號。你沒有持倉。Twitter 上人人都在喊「要崩了」。
  • A. 大量做多抄底！極度恐懼 = 買入機會
  • B. 跟著恐慌做空，趨勢明確
  • C. 等待：設定觀察清單，等確認信號（如 SuperTrend 翻多或吞沒K線）再行動
  • D. 用極小倉位試探做多，止損設在最低點下方
正確答案：C
「可能是底部」不等於「現在就是入場點」。在沒有確認信號的情況下抄底 = 接飛刀。正確做法：標記觀察價位，設定好 IF-THEN 規則（IF 出現反轉確認 THEN 小倉做多），然後等待。
A 的後果：「大量做多」在趨勢空頭中極度危險。BTC 可能再跌 15%。超賣可以更超賣。
B 的後果：在已經暴跌 8% 後追空 = 在恐慌最高點入場（同情緒循環中「投降」階段）。風報比極差。
D 的後果：這不是最差的選項，但「試探」容易變成「加倉」如果價格反彈一點。而且沒有確認信號的入場本質上是猜測。
心理學原理：從眾效應的反面。當所有人恐慌時你想逆勢，這需要等待確認而非盲目抄底。耐心是最高形式的紀律。
7沉沒成本的深淵
你一週前做多某山寨幣，已經虧了 25%。你的止損原本設在 -8% 但被你手動移除了。你告訴自己「它一定會回來」。現在項目的 TVL 在下降，社群活躍度也在降低。但你覺得：「都虧這麼多了，現在割肉太虧了。」
  • A. 立即平倉。問自己：「如果我現在空手，會在這個價格買入嗎？」答案是不會
  • B. 再等一週看看，也許有轉機
  • C. 加倉攤平，把均價拉低
  • D. 設一個新的止損在 -35%，給它最後機會
正確答案：A
關鍵問題：「如果我現在沒有持倉，以這個價格和這些基本面（TVL 下降、社群冷卻），我會買入嗎？」如果答案是否定的，你就不應該繼續持有。已經虧的 25% 是沉沒成本，不會因為你繼續持有而回來。
B 的後果：「再等一週」是損失厭惡+希望的組合。TVL 和社群數據都在惡化，基本面不支持反彈。
C 的後果：在基本面惡化的標的上加倉 = 往錯誤方向加碼。最危險的操作之一。
D 的後果：你已經移除過一次止損了。什麼讓你覺得到 -35% 時你不會再移一次？
心理學原理：沉沒成本謬誤 + 損失厭惡 + 承諾與一致性偏誤（既然當初決定持有，就繼續堅持「一致」）。
8盤中的無聊陷阱
你已經盯盤兩個小時了，今天的交易計劃中有 3 個設定，但都還沒有觸發入場條件。市場橫盤震盪，沒有方向。你開始感到焦躁：「難道今天就這樣白坐嗎？」你開始在更小的時間框架上找信號...
  • A. 切到 1 分鐘圖找入場機會，總能找到一些信號
  • B. 降低標準，那個「差一點」的信號也可以做
  • C. 找其他品種看有沒有機會
  • D. 關閉盤面去做其他事。今天沒有 A+ 設定就不交易
正確答案：D
「不交易也是一種策略」。你今天的計劃沒有觸發 = 市場今天不適合你的策略 = 不該交易。強迫交易的結果就是在低品質設定上虧錢，同時消耗手續費。離開螢幕去做有意義的事。
A 的後果：1 分鐘圖充滿噪音，在這個時間框架上找「信號」就像在噪音中聽音樂。手續費和滑點會吃掉你。
B 的後果：降低標準 = 降低勝率。你在用 B 級設定消耗 A 級的紀律。
C 的後果：如果其他品種在你的觀察清單內且有 A+ 設定，可以考慮。但如果是「為了交易而找東西做」，就是過度交易。
心理學原理：行動偏誤 + 多巴胺渴望。大腦把「行動」等同於「生產力」，把「等待」等同於「浪費時間」。但在交易中，等待才是最有生產力的行為。
9數據前的衝動
非農數據 10 分鐘後公布。你根據之前的分析判斷美元會走弱，已經想好了做多黃金的計劃。你想：「不如先佈局，等數據出來就能第一時間賺到。」
  • A. 提前入場，先佈局佔位
  • B. 等數據出來後 30 分鐘，看市場真正的反應再決定
  • C. 掛限價單在預期的支撐位
  • D. 數據方向明確後立即市價追入
正確答案：B
重大數據公布時的前 15-30 分鐘是最危險的時段。波動劇烈、滑點巨大、止損可能被跳空穿越。正確做法：等塵埃落定，看市場的「真正反應」（而非第一反應），然後按計劃入場。
A 的後果：如果數據意外強勁（美元走強），你在瞬間被反向止損，而且滑點可能讓虧損遠超預期。
C 的後果：數據公布時價格可能瞬間觸及你的限價單然後反轉，你被「假突破」的波動套住。
D 的後果：數據後的第一反應經常是假動作。「Buy the rumor, sell the news」。數據利多黃金但黃金反而跌的情況很常見。
心理學原理：「提前佈局」滿足的是控制幻覺和「聰明感」。你不需要是第一個入場的人，你需要是做對的人。
10大賺後的失控
你剛剛一筆交易賺了帳戶的 8%（平常一筆只賺 1-2%）。你極度興奮，覺得今天是你的「幸運日」。腎上腺素飆升，你想繼續做，趁手感好再開幾筆...
  • A. 繼續做！今天手感好就要多賺
  • B. 再做一筆，但用小倉位
  • C. 停手。大賺後的興奮 = 過度自信的前兆。今天到此為止
  • D. 休息 30 分鐘冷靜一下再說
正確答案：C
「興奮」是一種情緒狀態，不是交易信號。大賺後腎上腺素和多巴胺飆升，你的風險評估能力會顯著降低。統計顯示，大賺後立即再交易的勝率明顯低於平均。今天已經贏了，帶著利潤離開。
A 的後果：經典的「贏了還想贏」心態。你接下來的交易動機不是「好設定」而是「興奮感」。極可能把今天的利潤還回去。
B 的後果：「小倉位」不能解決你情緒過度興奮的問題。你的判斷力此刻是受損的。
D 的後果：休息 30 分鐘可能不夠讓腎上腺素完全消退。而且 30 分鐘後你可能找到理由繼續交易。
心理學原理：過度自信 + 多巴胺驅動的風險尋求行為。大腦在獲勝後會渴望「更多」，這和賭博機制完全相同。
11KOL 的魅力
一個你追蹤的知名交易員（50 萬粉絲）發文：「我全倉做多 SOL，目標 $300，這是歷史性的機會！」他之前確實有幾次很準的 call。你目前沒有自己對 SOL 的分析，但他說得很有信心...
  • A. 跟著做多，他這麼準應該可以信
  • B. 做多但用小倉位，止損設嚴一點
  • C. 先做自己的分析，如果結論一致再入場
  • D. 忽略。你不交易沒有經過自己分析確認的設定
正確答案：D
你不知道他的成本、他的止損、他的總倉位佔比、他是否有利益衝突（可能已經持有大量 SOL 需要散戶接盤）。跟單 = 把決策權交給不為你負責的人。沒有自己分析的交易 = 不做。
A 的後果：你不知道他什麼時候會平倉。他可能在你入場後就跑了。大 V 喊單 = 可能在出貨。
B 的後果：小倉位 + 嚴格止損降低了風險，但入場動機仍然是「跟單」不是「分析」。長期養成壞習慣。
C 的後果：這是次佳答案。但要誠實：你的「自己的分析」是否會被 KOL 的觀點污染？確認偏誤可能讓你只看到支持做多的證據。
心理學原理：從眾效應 + 權威偏誤 + 確認偏誤。「50 萬粉絲的人說的應該是對的」— 粉絲數不等於準確率。很多大 V 的真正收入來自流量，不是交易。
12破了規則但賺了錢
你昨天違反了自己的「連虧 3 筆停手」規則，在第 4 筆衝動下單做多 ETH — 沒有明確信號，倉位也比平常大。結果 ETH 意外暴漲，你這筆賺了帳戶的 5%。你想：「看吧，有時候跟著感覺走也是對的。」
  • A. 這是一筆「壞交易」即使賺了錢。記錄違規，不因結果而合理化
  • B. 結果好就好，不用太拘泥規則
  • C. 也許我的規則太嚴格了，應該放寬一些
  • D. 下次類似情況再試試看
正確答案：A
這是最危險的情況：違反紀律但獲利。這會讓大腦建立「違規 = 有獎勵」的神經連結，下次更容易違規。正確認知：好交易 = 遵守過程的交易（不管盈虧）。壞交易 = 違反過程的交易（即使賺錢）。
B 的後果：「結果好就好」= 典型的結果思維。這次賺了是運氣，下次同樣做法 90% 是虧。但你已經被「正強化」了。
C 的後果：一次違規的成功不應該改變經過深思的規則。你的規則是在冷靜理性時制定的，比情緒中的判斷更可靠。
D 的後果：= 確認了「跟著感覺走」。下次「類似情況」你會虧得更慘，因為你會用更大的信心（和倉位）去做。
心理學原理：正強化 + 結果偏誤。賭徒第一次贏錢最危險 — 因為它讓你覺得「這行得通」，然後投入更多直到破產。
